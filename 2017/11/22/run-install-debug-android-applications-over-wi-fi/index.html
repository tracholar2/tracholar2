没有根需要！

要查找设备的IP地址：运行adb shell，然后运行netcfg。你会在那里看到的。 要在使用OSX时查找IP地址，请运行命令adb shell ip route。

警告：启用该选项是危险的，即使您处于数据网络中，网络中的任何人都可以在调试中连接到您的设备。只有连接到可靠的Wi-
Fi时才可以使用，并且记得在完成时断开连接！
