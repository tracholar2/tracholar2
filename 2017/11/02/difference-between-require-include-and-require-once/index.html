require，include和require_once之间的区别？

By zhizi

Published 2017-11-02

Contents

在PHP中：

我应该什么时候使用require与include？ 我应该什么时候使用require_once与require？

还有require和include_once。

所以你的问题应该是…

我应该什么时候使用require与include？ 什么时候应该使用require_once与require

这里介绍1的答案。

The require() function is identical to include(), except that it handles
errors differently. If an error occurs, the include() function generates a
warning, but the script will continue execution. The require() generates a
fatal error, and the script will stop.

2的答案可以在这里找到。

The require_once() statement is identical to require() except PHP will check
if the file has already been included, and if so, not include (require) it
again.

使用

要求 当你的应用程序需要该文件时，例如一个重要的消息模板或一个包含配置变量的文件，没有这个应用程序将会中断。 require_once
