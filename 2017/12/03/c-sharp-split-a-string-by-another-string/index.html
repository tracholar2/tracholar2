C＃将字符串拆分为另一个字符串

By zhizi

发表于 2017-12-03

文章目录

我一直在使用Split()方法来拆分字符串，但是这只有在按字符拆分字符串时才有效。有没有办法拆分string，另一个字符串是由参数拆分？我已经尝试将分离器转换为字符数组，没有运气。

换句话说，我想拆分string：

THExxQUICKxxBROWNxxFOX

由xx，并返回一个数组值：

THE, QUICK, BROWN, FOX

为了分割一个字符串，你必须使用字符串数组重载。

string data = "THExxQUICKxxBROWNxxFOX";

return data.Split(new string[] { "xx" }, StringSplitOptions.None);

有一个需要字符串的Split的重载。

"THExxQUICKxxBROWNxxFOX".Split(new [] { "xx" }, StringSplitOptions.None);

您可以使用这些StringSplitOptions中的任何一个

无 - 返回值包含数组 包含空字符串的元素 RemoveEmptyEntries - 返回值 不包含数组元素 包含一个空字符串

所以如果字符串是“THExxQUICKxxxxBROWNxxFOX”，那么StringSplitOptions.None将在数组中为“xxxx”部分返回一个空的条目，而StringSplitOptions.RemoveEmptyEntries则不会。
