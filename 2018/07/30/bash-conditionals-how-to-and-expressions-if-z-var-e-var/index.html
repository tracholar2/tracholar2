bash条件：如何“和”表达式？ （如果[！-z $ VAR && -e $ VAR]）

By zhizi

Published 2018-07-30

Contents

我想我不清楚如何做“和”测试。我想确保存在一个与[-e $ VAR]一致的参数，但事实证明在空字符串上也被评估为true;我不想要。

我怎么和他们在一起？或者还有另一个完成我想要的测试吗？

if [ ! -z "$var" ] && [ -e "$var" ]; then
      # something ...
fi

从bash手册页：

[[ expression ]] - return a status of 0 or 1 depending on the evaluation
of the conditional expression expression.

而对于表达式，其中一个选项是：

expression1 && expression2 - true if both expression1 and expression2are true.

所以你可以把和放在一起，如下所示（-n与-z是相反的，所以我们可以去掉` ！> ）：

if [[ -n "$var" && -e "$var" ]] ; then
    echo "'$var' is non-empty and the file exists"
fi

然而，在这种情况下，我不认为这是需要的，如果xyzzy 文件 存在，-e xyzzy是真的，轻松处理空弦。如果这就是你想要的，那么你实际上并不需要-z非空检查：

pax> VAR=xyzzy
pax> if [[ -e $VAR ]] ; then echo yes ; fi
pax> VAR=/tmp
pax> if [[ -e $VAR ]] ; then echo yes ; fi
yes

换句话说，就是使用：

if [[ -e "$var" ]] ; then
    echo "'$var' exists"
fi
