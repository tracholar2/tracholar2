我花了几个月的时间来掌握WCF背后的概念，最近我开发了我的第一个WCF服务应用程序。
为了理解配置文件中的所有设置，我努力了很多 我不相信环境，但似乎你可以用它做很棒的事情 有一天，我发现微软已经出现了一个叫做 ASP.NET Web
API 的新东西 对于我所能读的，它是一个 RESTful框架 ，非常易于使用和实现
现在，我想弄清楚这两个框架之间的主要区别是什么，如果我尝试使用新的API来转换我的旧的_ WCF服务应用程序。

请有人帮助我了解每个人的差异和用法吗？
