如何从shell脚本获取密码而不回显

By zhizi

发表于 2018-06-23

文章目录

我有一个脚本可以自动执行需要访问密码保护系统的进程。系统通过一个接受用户密码作为参数的命令行程序来访问。

我想提示用户输入他们的密码，把它分配给一个shell变量，然后使用这个变量来构造访问程序的命令行（这当然会产生我将要处理的流输出） 。

我是Bourne / Bash的一名合理的shell编程人员，但是我不知道如何接受用户输入，而不用回应终端（或者用“*”字符回显）。

任何人都可以帮忙吗？

这是另一种方法：

#!/bin/bash
# Read Password
echo -n Password: 
read -s password
echo
# Run Command
echo $password

read -s会关闭你的echo。只需将最后一行的echo替换为您要运行的命令

即可

#!/bin/bash
stty -echo
printf "Password: "
read PASSWORD
stty echo
printf "\n"
