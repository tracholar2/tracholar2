如何检查一个数是否是2的幂

By zhizi

Published 2018-06-06

Contents
  1. 1. Explanation

今天，我需要一个简单的算法来检查一个数是2的幂。

该算法需要：

  • 简单
  • 更正所有ulong值。

我想出了这个简单的算法：

private bool IsPowerOfTwo(ulong number)
{
    if (number == 0)
        return false;

    for (ulong power = 1; power > 0; power = power << 1)
    {
        // This for loop used shifting for powers of 2, meaning
        // that the value will become 0 after the last shift
        // (from binary 1000...0000 to 0000...0000) then, the 'for'
        // loop will break out.

        if (power == number)
            return true;
        if (power > number)
            return false;
    }
    return false;
}

但后来我想，如果检查log2 x是否恰好是一个整数？但是当我检查2 ^ 63 + 1时，由于四舍五入，Math.Log
完全返回63。所以我检查了2的幂63是否等于原来的数字 - 这是，因为计算是在 double `中完成的，而不是确切的数字：

private bool IsPowerOfTwo_2(ulong number)
{
    double log = Math.Log(number, 2);
    double pow = Math.Pow(2, Math.Round(log));
    return pow == number;
}

对于给定的错误值，这返回了true：9223372036854775809。

有更好的算法吗？

这个问题有一个简单的技巧：

bool IsPowerOfTwo(ulong x)
{
    return (x & (x - 1)) == 0;
}

注意，这个函数将为0报告true，这不是2的威力。如果你想排除，那么这是如何：

bool IsPowerOfTwo(ulong x)
{
    return (x != 0) && ((x & (x - 1)) == 0);
}

Explanation

首先是按位二进制＆amp;来自MSDN定义：

Binary & operators are predefined for the integral types and bool. For
integral types, & computes the logical bitwise AND of its operands. For bool
operands, & computes the logical AND of its operands; that is, the result is
true if and only if both its operands are true.

现在让我们来看看这一切如何发挥：

该函数返回布尔值（true / false），并接受一个类型为unsigned
long（x的输入参数，在这种情况下）。让我们为了简单起见，假设有人已经通过了值4，并调用这样的函数：

bool b = IsPowerOfTwo(4)

现在我们用4替换每个x：

return (4 != 0) && ((4 & (4-1)) == 0);

好吧，我们已经知道4！= 0是真实的，至今为止都是如此。但是怎么样：

((4 & (4-1)) == 0)

这当然可以翻译成：

((4 & 3) == 0)

但究竟是4＆amp; 3？

4的二进制表示是100，而3的二进制表示是011（记住＆amp;将这些数字的二进制表示法，所以我们有：

100 = 4
011 = 3

想像这些价值叠加起来很像基本的加法。 ＆amp;运算符指出，如果两个值都等于1，那么结果是1，否则它是0.所以1＆amp; 1 = 1，1＆amp; 0 = 0，0＆amp; 0 = 0和0＆amp; 1 = 0。所以我们做数学：

100
011
----
000

结果是简单的0.所以我们回头看看我们的返回语句现在转化为：

return (4 != 0) && ((4 & 3) == 0);

现在翻译成：

return true && (0 == 0);



return true && true;

大家都知道true＆amp;＆amp; true简单地就是true，这表明对于我们的例子来说，4是2的幂。
