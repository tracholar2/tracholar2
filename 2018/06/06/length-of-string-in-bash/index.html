如何获取存储在变量中的字符串长度并将其分配给另一个变量？

myvar="some string"
echo ${#myvar}  
# 11

你如何设置另一个变量到输出11？

Length of an argument

参数与常规变量相同

    local bytlen sreal oLang=$LANG oLcAll=$LC_ALL

要获取存储在变量中的字符串的长度，请说：

myvar="some string"
size=${#myvar}

要确认它被正确保存，echo它：

$ echo "$size"
