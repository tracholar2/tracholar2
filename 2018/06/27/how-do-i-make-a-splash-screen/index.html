如何 ：简单的启动画面

这个答案显示了当您的应用程序启动时，如何在固定的时间内显示启动画面。品牌的原因。例如。你可以选择显示启动画面3秒钟。但是，如果你想显示spash屏幕的时间可变（例如应用程序启动时间），你应该检查出Abdullah的答案https：/ /stackoverflow.com/a/15832037/401025。不过请注意，在新设备上启动应用的速度可能非常快，所以用户只会看到一个不好的用户体验的闪光灯。

首先，您需要在您的layout.xml文件

中定义启动画面

<?xml version="1.0" encoding="utf-8"?>
<LinearLayout xmlns:android="http://schemas.android.com/apk/res/android"
        android:orientation="vertical" android:layout_width="fill_parent"
        android:layout_height="fill_parent">

        <ImageView android:id="@+id/splashscreen" android:layout_width="wrap_content"
                android:layout_height="fill_parent"
                android:src="@drawable/splash"
                android:layout_gravity="center"/>

        <TextView android:layout_width="fill_parent"
                  android:layout_height="wrap_content"
                  android:text="Hello World, splash"/>

</LinearLayout>

还有你的活动：

import android.app.Activity;
import android.content.Intent;
import android.os.Bundle;
import android.os.Handler;

public class Splash extends Activity {

    /** Duration of wait **/
    private final int SPLASH_DISPLAY_LENGTH = 1000;

    /** Called when the activity is first created. */
    @Override
    public void onCreate(Bundle icicle) {
        super.onCreate(icicle);
        setContentView(R.layout.splashscreen);

        /* New Handler to start the Menu-Activity 
         * and close this Splash-Screen after some seconds.*/
        new Handler().postDelayed(new Runnable(){
            @Override
            public void run() {
                /* Create an Intent that will start the Menu-Activity. */
                Intent mainIntent = new Intent(Splash.this,Menu.class);
                Splash.this.startActivity(mainIntent);
                Splash.this.finish();
            }
        }, SPLASH_DISPLAY_LENGTH);
    }
}

这是所有的;）

注意这个解决方案不会让用户等待更多：**启动画面的延迟取决于应用程序的启动时间。

当你打开任何android应用程序时，你会得到一个黑色的屏幕，上面有应用程序的标题和图标，你可以使用style / theme来改变它。

首先，在values文件夹中创建一个style.xml并为其添加一个样式。

<style name="splashScreenTheme" parent="@android:style/Theme.DeviceDefault.Light.NoActionBar">
    <item name="android:windowBackground">@drawable/splash_screen</item>
</style>

使用@android：style / Theme.DeviceDefault.Light.NoActionBar，您可以使用任何其他主题作为父项。

其次，在您的应用程序Manifest.xml中为您的主要活动添加android：theme =“@ style / splashScreenTheme”。

<activity
        android:name="MainActivity"
        android:label="@string/app_name"
        android:theme="@style/splashScreenTheme" >

第三，在你的onCreate（）启动活动中更新你的主题。

protected void onCreate(Bundle savedInstanceState) {
    // Make sure this is before calling super.onCreate
    setTheme(R.style.mainAppTheme);
    super.onCreate(savedInstanceState);
}
