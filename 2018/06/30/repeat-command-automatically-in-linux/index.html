在Linux中自动重复命令

By zhizi

发表于 2018-06-30

文章目录

是否可以在linux命令行中每n秒重复一次命令。

例如，说我有一个导入运行，我正在做

ls -l

检查文件大小正在增加。我想要一个命令来自动重复。

每5秒观看一次…

` watch -n 5 ls -l <​​/ code>

如果您希望对更改进行可视化确认，请在ls命令之前附加\- differences。

根据OSX手册页，还有

The –cumulative option makes highlighting “sticky”, presenting a running
display of all positions that have ever changed. The -t or –no-title option
turns off the header showing the interval, command, and current time at the
top of the display, as well as the following blank line.

Linux / Unix手册页可以在这里找到 

while true; do
    sleep 5
    ls -l
done
