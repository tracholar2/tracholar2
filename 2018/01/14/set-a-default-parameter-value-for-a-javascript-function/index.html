为JavaScript函数设置默认参数值

By zhizi

发表于 2018-01-14

文章目录

我想要一个JavaScript函数有可选的参数，我设置了一个默认值，如果这个值没有被定义的话会被使用。在Ruby中，你可以这样做：

def read_file(file, delete_after = false)
  # code
end

这是否在JavaScript中工作？

function read_file(file, delete_after = false) {
  // Code
}

在ES6 / ES2015中，默认参数在语言规范中。

function read_file(file, delete_after = false) {
  // Code
}

正常。

function read_file(file, delete_after) {
    delete_after = delete_after || "my default here";

如果delete_after的值不是 _值，那么它将赋值给delete_after，否则将赋予字符串`“我的默认值这里“。有关更多详细信息，请查看 Doug
Crockford对语言的调查，并查看有关运营商的部分。

如果您要传递 falsey 值，即false，null，undefined，0或“”。如果您需要传递falsey值，则需要使用 Tom
Ritter的答案。

在处理函数的许多参数时，允许使用者传递对象中的参数参数，然后将这些值与包含默认值的对象合并_为功能

function read_file(values) {
    values = merge({ 
        delete_after : "my default here"
    }, values || {});

    // rest of code
}

// simple implementation based on $.extend() from jQuery
function merge() {
    var obj, name, copy,
        target = arguments[0] || {},
        i = 1,
        length = arguments.length;

    for (; i < length; i++) {
        if ((obj = arguments[i]) != null) {
            for (name in obj) {
                copy = obj[name];

                if (target === copy) {
                    continue;
                }
                else if (copy !== undefined) {
                    target[name] = copy;
                }
            }
        }
    }

    return target;
};

使用

// will use the default delete_after value
read_file({ file: "my file" }); 

// will override default delete_after value
read_file({ file: "my file", delete_after: "my value" });
