您可以使用 重新加载模块> reload Python 2中的内建函数：

import foo

while True:
    # Do some things.
    if is_changed(foo):
        foo = reload(foo)

在Python 3中，reload被移动到 imp模块。在3.4中，imp已被弃用，以支持 `importlib < / code> 和 reload被添加到后者。定位3或更高版本时，请在调用reload时引用适当的模块或导入它。

我想这就是你想要的。像Django的开发服务器这样的Web服务器使用它，这样你就可以看到你的代码更改的效果，而不用重新启动服务器进程本身。
