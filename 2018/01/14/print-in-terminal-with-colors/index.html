在颜色的终端打印？

By zhizi

Published 2018-01-14

Contents

如何在Python中输出彩色文本到终端？ 代表固体块的最佳Unicode符号是什么？

这有点取决于你在哪个平台上。最常见的方法是打印ANSI转义序列。举个简单的例子，下面是一些来自搅拌机构建脚本：

class bcolors:
    HEADER = '\033[95m'
    OKBLUE = '\033[94m'
    OKGREEN = '\033[92m'
    WARNING = '\033[93m'
    FAIL = '\033[91m'
    ENDC = '\033[0m'
    BOLD = '\033[1m'
    UNDERLINE = '\033[4m'

要使用这样的代码，你可以做一些像

的事情

print bcolors.WARNING + "Warning: No active frommets remain. Continue?" 
      + bcolors.ENDC

这将适用于unix，包括OS X，Linux和Windows（只要您使用 ANSICON，或Windows 10提供了启用VT100仿真的功能。有用于设置颜色，移动光标等的ansi代码。

如果你打算使这个变得复杂（而且听起来像是你正在编写一个游戏），你应该看看“curses”模块，它为你处理了很多复杂的部分。 Python Curses
HowTO 是一个很好的介绍。

如果你没有使用扩展的ASCII码（即不在PC上），那么你将被扣上127以下的ascii字符，’＃’或’@’可能是你最好的选择。如果您可以确保您的终端正在使用IBM
扩展ascii字符集，则还有更多选项。字符176,177,178和219是“块字符”。

一些现代的基于文本的程序，如“矮人堡垒”，在图形模式下模拟文本模式，并使用经典PC字体的图像。你可以在矮人要塞维基上找到一些这样的位图，参见（用户制作的图集）。

文字模式演示比赛有更多的资源可供选择在文本模式下的图形。

嗯..我觉得这个答案有点让人失望。虽然我正在计划一个史诗般的基于文本的冒险游戏。祝你好运彩色文字！

我很惊讶没有人提到 Python termcolor模块。用法很简单：

from termcolor import colored

print colored('hello', 'red'), colored('world', 'green')

然而，对于游戏编程和你想要做的“彩色块”来说，它可能还不够成熟……
