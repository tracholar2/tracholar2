在JavaScript中定义枚举的首选语法是什么？

By zhizi

Published 2018-01-14

Contents

在JavaScript中定义枚举的首选语法是什么？就像：

my.namespace.ColorEnum = {
    RED : 0,
    GREEN : 1,
    BLUE : 2
}

// later on

if(currentColor == my.namespace.ColorEnum.RED) {
   // whatever
}

还是有更好的成语？

我无法发表评论给答案，所以我想我会碰到线索，因为它显示在谷歌高。

由于1.8.5可以封闭和冻结对象，因此将上面定义为：

var DaysEnum = Object.freeze({"monday":1, "tuesday":2, "wednesday":3, ...})

或

var DaysEnum = {"monday":1, "tuesday":2, "wednesday":3, ...}
Object.freeze(DaysEnum)

瞧！ JS枚举;）

来源： https://developer.mozilla.org/en/JavaScript/Reference/Global_Objects
/对象/冻结

恕我直言，引号是不需要的，但我保持一致。

这不是一个很好的答案，但是我会说这很好，亲自

话虽如此，因为它的值是什么都没有关系（你已经使用了0,1,2），所以如果你想要输出当前值的话，我会使用一个有意义的字符串。
