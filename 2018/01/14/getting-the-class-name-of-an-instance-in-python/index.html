在Python中获取实例的类名

By zhizi

Published 2018-01-14

Contents

我如何找到一个在Python中创建了一个对象实例的类的名字，如果我这样做的函数是这个实例的类的派生类的基类？

正在考虑也许检查活物检查模块< /
a>可能帮助我在这里，但它似乎并没有给我我想要的。而且，对__ class __成员解析不够，我不确定如何获取这些信息。

您是否尝试过类的__ name __属性？即type（x）.__ name __会给你这个类的名字，我想是你想要的。

>>> import itertools
>>> x = itertools.count(0)
>>> type(x).__name__
'count'

此方法仅适用于新式课程。你的代码可能会使用一些旧式的类。以下内容适用于两者：

x.__class__.__name__

你想把这个类的名字当作一个字符串吗？

instance.__class__.__name__
