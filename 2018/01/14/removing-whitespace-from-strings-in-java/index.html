在Java中删除字符串中的空格

By zhizi

Published 2018-01-14

Contents

我有这样的字符串：

mysz = "name=john age=13 year=2001";

我想删除字符串中的空格。我试过trim（），但是这样只删除整个字符串前后的空格。我也尝试了replaceAll（“\\ W”，“”），但是=也被移除了。

我怎样才能得到一个字符串：

mysz2 = "name=johnage=13year=2001"

st.replaceAll（“\\ s +”，“”）删除所有的空格和不可见的字符（例如tab， \ n `）。

>

st.replaceAll（“\ s +”，“”）和 st.replaceAll（“\ s”，“”）`会产生相同的结果。 / p>
第二个正则表达式比第一个正则表达式快20％，但是随着连续空格数的增加，第一个正则表达式比第二个正则表达式要好。

如果不是直接使用，则将值赋给变量：

st = st.replaceAll("\\s+","")



replaceAll("\\s","")

\ w =任何单词字符

\ W =任何不是单词的字符（包括标点符号等）

\ s =任何空格字符（包括空格，制表符等）

\ S =任何不是空格的字符（包括字母和数字，以及标点符号等）

（编辑：正如你指出的，如果你想要\ s到达正则表达式引擎，你需要转义反斜杠，导致\\ s。）
