如何从int转换为String？

By zhizi

Published 2018-01-14

Contents

我正在开发一个项目，从int到String的所有转换都是这样完成的：

int i = 5;
String strI = "" + i;

我不熟悉Java。按照我的想法，这是通常的做法还是错误的？

正常的方式是Integer.toString（i）或String.valueOf（i）。

这个连接是可行的，但它是非常规的，可能是一种难闻的气味，因为它表明作者不知道上述两种方法（他们还不知道什么？）。

在使用字符串时，Java对+运算符有特别的支持（参见文档），它将您发布的代码翻译成：

StringBuilder sb = new StringBuilder();
sb.append("");
sb.append(i);
String strI = sb.toString();

在编译时。它的效率稍低（sb.append（））最终调用Integer.getChars（），这就是Integer.toString（）不管怎么说），但它的工作原理。

回答Grodriguez的评论：**不，在这种情况下，编译器 _不会优化空字符串 - 看起来：

simon@lucifer:~$ cat TestClass.java
public class TestClass {
  public static void main(String[] args) {
    int i = 5;
    String strI = "" + i;
  }
}

提案和正在进行的工作来改变这种行为，针对JDK 9。

这是可以接受的，但我从来没有写过类似的东西。我更喜欢这个：

String strI = Integer.toString(i);
