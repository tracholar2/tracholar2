在jQuery中添加表格行

By zhizi

Published 2018-01-14

Contents

jQuery中最好的方法是将最后一行添加到表中作为最后一行？

这是可以接受的吗？

$('#myTable').append('<tr><td>my data</td><td>more data</td></tr>');

是否有限制，你可以添加到这样的表（如输入，选择，行数）？

您建议的方法不能保证给出您要查找的结果 - 例如，如果您有tbody，例如：

<table id="myTable">
  <tbody>
    <tr>...</tr>
    <tr>...</tr>
  </tbody>
</table>

你会得到以下结果：

<table id="myTable">
  <tbody>
    <tr>...</tr>
    <tr>...</tr>
  </tbody>
  <tr>...</tr>
</table>

因此，我会推荐这种方法：

$('#myTable tr:last').after('<tr>...</tr><tr>...</tr>');

只要是有效的HTML，就可以在 after（）`方法中包含任何内容，包括上面例子中的多行。

更新： 在此问题的最近活动之后重新访问此答案。无瑕的眼神使得一个很好的评论，DOM中总是会有一个tbody这是真的，但只有至少有一行。如果您没有行，除非您自己指定了行，否则不会有tbody。

DaRKoN_ 建议附加到tbody而不是在最后一个tr之后添加内容。这样可以解决没有行的问题，但是仍然不是无懈可击的，因为理论上你可以有多个tbody元素，行会被添加到每个元素中。

称重一切，我不确定是否有单一的解决方案来解决每一个可能的情况。您需要确保jQuery代码符合您的标记。

我认为最安全的解决方案可能是确保您的table在您的标记中至少包含一个tbody，即使它没有行。在这个基础上，你可以使用下面的代码，但是你可以使用很多行（也可以是多个tbody元素）：

$('#myTable > tbody:last-child').append('<tr>...</tr><tr>...</tr>');

jQuery有一个内置的工具，可以随时操作DOM元素。

您可以像这样为您的表添加任何内容：

$("#tableID").find('tbody')
    .append($('<tr>')
        .append($('<td>')
            .append($('<img>')
                .attr('src', 'img.png')
                .text('Image cell')
            )
        )
    );

jQuery中的$（'＆lt; some-tag＆gt;'）是一个标签对象，可以有多个attr属性可以设置和获取，以及text，它表示标签之间的文本：＆lt; tag＆gt; text＆lt; / tag＆gt;。

这是一些非常奇怪的缩进，但是在这个例子中看起来更容易。
