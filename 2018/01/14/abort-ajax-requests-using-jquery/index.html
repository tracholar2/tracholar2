使用jQuery中止Ajax请求

By zhizi

发表于 2018-01-14

文章目录

使用jQuery，如何 取消/中止Ajax请求 ，我还没有收到回应？

大多数jQuery Ajax方法都返回一个XMLHttpRequest（或者等价的）对象，所以你可以使用abort（）。

请参阅文档：

  • 放弃方法（ MSDN ）。取消当前的HTTP请求。
  • abort（）（ MDN ）。如果请求已经发送，则此方法将中止请求。
var xhr = $.ajax({
    type: "POST",
    url: "some.php",
    data: "name=John&location=Boston",
    success: function(msg){
       alert( "Data Saved: " + msg );
    }
});

//kill the request
xhr.abort()

<强>更新 从jQuery
1.5开始，返回的对象是一个名为jqXHR的本地XMLHttpRequest对象的包装。这个对象似乎公开了所有的本地属性和方法，所以上面的例子仍然有效。请参阅
jqXHR Object （jQuery API文档）。<
/ p>

更新2： 从jQuery 3开始，ajax方法返回一个没有额外方法（如中止）的promise，所以这将不再工作。请参阅此处的3.0博客。一个办法是通过xhr属性来控制xhr对象。这是一个简单的例子：

var xhr = new window.XMLHttpRequest();
var request = $.ajax({
    url : url,
    xhr : function(){
        return xhr;
    }
});
xhr.abort();

您无法记起请求，但可以设置一个超时值，之后响应将被忽略。查看这个页面查询jQuery
AJAX选项。我相信，如果超时超时，你的错误回调将被调用。每个AJAX请求都有一个默认超时。

您也可以使用abort（）方法请求对象，但是，虽然它会导致客户端停止监听事件，但是
可能 可能不会阻止服务器处理它。
