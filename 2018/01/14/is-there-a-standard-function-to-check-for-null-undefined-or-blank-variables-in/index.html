有没有一个标准的函数来检查JavaScript中的空，未定义或空白变量？

By zhizi

Published 2018-01-14

Contents

是否有一个通用的JavaScript函数检查变量是否有值，并确保它不是undefined或null？我有这个代码，但我不确定它是否涵盖所有情况：

function isEmpty(val){
    return (val === undefined || val == null || val.length <= 0) ? true : false;
}

您可以检查 变量 是否包含truthy值。这意味着

if( value ) {
}

如果value是 not ：

将评估为true：

  • 空
  • 未定义
  • 的NaN
  • 空字符串（“”）
  • 0
  • 假

上面的列表代表ECMA- / Javascript中的所有可能的falsy值。在[规范](http://www.ecma- international.org/publications/files/ECMA-ST/ECMA-262.pdf)中找到它> ToBoolean部分。

此外，如果你不知道变量是否存在（也就是说，如果是声明的），你应该检查typeof运营商。比如

if( typeof foo !== 'undefined' ) {
    // foo could get resolved and it's defined
}

如果您确定至少声明了一个 变量 ，那么您应该直接检查它是否具有如上所示的truthy值。

进一步阅读： http://typeofnan.blogspot.com/2011 /01/typeof-is-fast.html

// value is undefined or null
return value === undefined || value === null;

JavaScript忍者可以使用 == 运算符：

return value == null;
