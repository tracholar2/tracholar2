package.json中的tilde（〜）和caret（^）有什么区别？

By zhizi

Published 2018-01-14

Contents

在升级到最新的node和npm之后，我尝试了npm install moment --save。它使用caret（^）前缀将条目保存在package.json中。以前，这是一个tilde（〜）前缀。

  • 为什么在npm中进行这些更改？
  • 代字号（〜）和 caret（^）`有什么区别？
  • 有什么优势？

In the simplest terms, the tilde matches the most recent minor version (the
middle number). ~1.2.3 will match all 1.2.x versions but will miss 1.3.0.

>

The caret, on the other hand, is more relaxed. It will update you to the
most recent major version (the first number). ^1.2.3 will match any 1.x.x
release including 1.3.0, but will hold off on 2.0.0.

http://fredkschott.com/post / 2014/02 / NPM-不再视为默认到颚化符号/
