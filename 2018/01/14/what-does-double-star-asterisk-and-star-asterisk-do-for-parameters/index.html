（双星/星号）和（星号/星号）为参数做了什么？

By zhizi

Published 2018-01-14

Contents

在下面的方法定义中，和**为param2做了什么`？

def foo(param1, *param2):
def bar(param1, **param2):

* args和** kwargs是一个常用的习惯用法，它允许函数的任意数量的参数，如[关于在Python文档中定义函数的更多信息。

* args会给你所有的函数参数[作为元组](“https://docs.python.org/dev/tutorial/controlflow.html

#arbitrary-argument-lists”rel)：

In [1]: def foo(*args):
   ...:     for a in args:
   ...:         print a
   ...:         
   ...:         

In [2]: foo(1)
1


In [4]: foo(1,2,3)
1
2
3

** kwargs会给你所有的 关键字参数 除了那些与形式参数相对应的字典。

In [5]: def bar(**kwargs):
   ...:     for a in kwargs:
   ...:         print a, kwargs[a]
   ...:         
   ...:         

In [6]: bar(name='one', age=27)
age 27
name one

这两个习语都可以和正常的参数混合在一起，以允许一组固定的和一些可变的参数：

def foo(kind, *args, **kwargs):
   pass

另一个使用* l成语的方法是在调用函数时 解开参数列表 。

In [9]: def foo(bar, lee):
   ...:     print bar, lee
   ...:     
   ...:     

In [10]: l = [1,2]

In [11]: foo(*l)
1 2

在Python 3中，可以在任务的左侧使用* l（ Extended Iterable Unpacking），尽管它在这个上下文中给出了一个列表而不是一个元组：

first, *rest = [1,2,3,4]
first, *l, last = [1,2,3,4]

另外，Python 3增加了新的语义（请参阅 PEP 3102）：< / p>

def func(arg1, arg2, arg3, *, kwarg1, kwarg2):
    pass

这个函数只接受3个位置参数，*后面的所有内容只能作为关键字参数传递。

值得一提的是，在调用函数时也可以使用*和**。这是一个快捷方式，允许您直接使用列表/元组或字典将多个参数传递给函数。例如，如果您有以下功能：

def foo(x,y,z):
    print("x=" + str(x))
    print("y=" + str(y))
    print("z=" + str(z))

你可以做这样的事情：

>>> mylist = [1,2,3]
>>> foo(*mylist)
x=1
y=2
z=3

>>> mydict = {'x':1,'y':2,'z':3}
>>> foo(**mydict)
x=1
y=2
z=3

>>> mytuple = (1, 2, 3)
>>> foo(*mytuple)
x=1
y=2
z=3

注意：mydict中的键必须与函数foo的参数完全一致。否则，它会抛出一个TypeError。
