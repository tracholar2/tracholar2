Java字符串到日期的转换

By zhizi

Published 2018-01-14

Contents
  1. 1. Java 8 update

有人可以推荐最好的方式来转换格式’2010年1月2日’的字符串在Java中的日期？最终，我想把整个月份，每一天，每一年都分解为整数，以便我可以使用：

Date date = new Date();
date.setMonth()..
date.setYear()..
date.setDay()..
date.setlong currentTime = date.getTime();

将日期转换成时间。

不要这样做，这是困难的方式。此外，自Java 1.1（1997）以来，那些java.util.Date的setter方法就被弃用了。
简单地使用SimpleDateFormat格式化日期匹配输入字符串的格式模式。

在你的具体情况下，“2010年1月2日”作为输入字符串，“1月”是全文月份，所以使用MMMM模式，“2”是短日期-month，所以使用d模式，“2010”是4位数的年份，所以使用yyyy模式。

String string = "January 2, 2010";
DateFormat format = new SimpleDateFormat("MMMM d, yyyy", Locale.ENGLISH);
Date date = format.parse(string);
System.out.println(date); // Sat Jan 02 00:00:00 GMT 2010

注意显式Locale参数的重要性。如果你忽略它，那么它将使用默认语言环境，不一定是英文，如输入字符串的月份名称中使用的。如果语言环境与输入字符串不匹配，那么即使格式模式似乎有效，也会混淆得到一个java.text.ParseException。

重要提示：SimpleDateFormat是 不是

Java 8 update

如果您碰巧已经在Java 8上，请使用 DateTimeFormatter（也可以在这里单击链接查看所有预定义的格式化程序和可用的格式模式;
[本教程可在这里找到）。这个新的API受到 JodaTime 的启发。

String string = "January 2, 2010";
DateTimeFormatter formatter = DateTimeFormatter.ofPattern("MMMM d, yyyy", Locale.ENGLISH);
LocalDate date = LocalDate.parse(string, formatter);
System.out.println(date); // 2010-01-02

注意：如果你的格式模式恰好也包含时间部分，那么使用 LocalDateTime＃parse（text，formatter）而不是< code> LocalDate＃parse（text，formatter）` 。而且，如果您的格式模式恰好也包含时区，请使用 ZonedDateTime＃parse（text，formatter）。
