JavaScript相当于printf / string format

By zhizi

发表于 2018-01-14

文章目录

我正在寻找一个与C / PHP printf（）或C＃/ Java程序员相当的JavaScript代码，String.Format（）（`IFormatProvider for .NET）。

我的基本需求现在是数字分隔格式，但是处理大量组合（包括日期）的东西会很好。

我意识到微软的 Ajax 库提供了一个String的版本。 Format（），但我们不希望整个框架的开销。

请试用 sprintf（）for JavaScript 。

好的，如果你真的想自己做一个简单的格式化方法，不要连续进行替换，而是同时进行替换。

由于大多数提及的其他建议都失败，因为以前替换的替换字符串也包含这样的格式序列：

"{0}{1}".format("{1}", "{0}")

通常你会希望输出是{1} {0}，但实际输出是{1} {1}。所以做一个同时替换，而不是像fearphage的建议。 p> 建立在以前建议的解决方案上：

// First, checks if it isn't implemented yet.
if (!String.prototype.format) {
  String.prototype.format = function() {
    var args = arguments;
    return this.replace(/{(\d+)}/g, function(match, number) { 
      return typeof args[number] != 'undefined'
        ? args[number]
        : match
      ;
    });
  };
}

格式（“ASP”，“ASP.NET”）“{0} is dead，but {1} is alive！{0} {2}

输出

ASP is dead, but ASP.NET is alive! ASP {2}

如果您不想修改String的原型：

if (!String.format) {
  String.format = function(format) {
    var args = Array.prototype.slice.call(arguments, 1);
    return format.replace(/{(\d+)}/g, function(match, number) { 
      return typeof args[number] != 'undefined'
        ? args[number] 
        : match
      ;
    });
  };
}

让你更熟悉：

String.format（'{0} is dead，but {1} is alive！{0} {2}'，'ASP'，'ASP.NET'）;
结果相同：

ASP is dead, but ASP.NET is alive! ASP {2}
