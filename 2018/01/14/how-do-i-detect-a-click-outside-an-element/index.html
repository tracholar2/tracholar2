如何检测元素外的点击？

By zhizi

Published 2018-01-14

Contents
  1. 1. Edit a 2017-06-23

我有一些HTML菜单，当用户点击这些菜单的头部时，我会完全显示这些菜单。当用户点击菜单区域外时，我想要隐藏这些元素。

这是jQuery可能的事情吗？

$("#menuscontainer").clickOutsideThisElement(function() {
    // Hide the menus
});

NOTE: Using stopEventPropagation() is something that should be avoided as
it breaks normal event flow in the DOM. See this article for more information. Consider
using this method instead.

将一个点击事件附加到关闭窗口的文档主体。将一个单独的点击事件附加到停止传播到文档正文的窗口。

$(window).click(function() {
//Hide the menus if visible
});

$('#menucontainer').click(function(event){
    event.stopPropagation();
});

您可以在document上监听 点击 事件，然后确保#menucontainer不是始祖或点击元素的目标通过使用 .closest（） 。

如果不是，那么被点击的元素在#menucontainer之外，您可以安全地隐藏它。

$(document).click(function(event) { 
    if(!$(event.target).closest('#menucontainer').length) {
        if($('#menucontainer').is(":visible")) {
            $('#menucontainer').hide();
        }
    }        
});

Edit a 2017-06-23

如果您打算关闭菜单并且想要停止监听事件，您也可以在事件监听器之后进行清理。该函数只会清理新创建的侦听器，保留document上的其他任何点击侦听器。使用ES2015语法：

export function hideOnClickOutside(selector) {
  const outsideClickListener = (event) => {
    if (!$(event.target).closest(selector).length) {
      if ($(selector).is(':visible')) {
        $(selector).hide()
        removeClickListener()
      }
    }
  }

  const removeClickListener = () => {
    document.removeEventListener('click', outsideClickListener)
  }

  document.addEventListener('click', outsideClickListener)
}
