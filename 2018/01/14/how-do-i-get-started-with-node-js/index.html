首先，学习Node.js的核心概念：**

  • 你会想了解Node.js鼓励的异步编码风格。

  • 异步！=并发。了解Node.js的事件循环！

  • [ Node.js使用CommonJS风格的require（）代码加载;这可能与您习惯的有所不同。

  • 熟悉Node.js的标准库。

然后，你将要看看社区提供什么：**

Node.js包管理的黄金标准是 NPM 。

  • 这是一个管理你的命令行工具项目的依赖关系。

  • 确保您了解Node.js和NPM如何通过node_modules文件夹与您的项目进行交互的package.json 。

  • NPM也是几乎所有Node.js包的注册表

最后，你会想知道一些比较流行的软件包是用于各种任务的：**

每个项目的有用工具：

  • 下划线包含您想要的每种核心实用程序方法。
  • Lo-Dash 是Underscore的一个克隆，其目标是更快，更具可定制性，并且具有不少功能下划线没有。它的某些版本可以用作下划线的直接替换。
  • TypeScript 使得JavaScript变得更加可以忍受，同时也让你免于麻烦！
  • JSHint 是一个代码检查工具，可以帮您节省大量的时间，找到愚蠢的错误。找到您的文本编辑器的插件，将自动在您的代码上运行它。

单元测试：

  • 摩卡是一个流行的测试框架。
  • Vows 对于异步测试来说是一个奇妙的尝试，尽管有点陈旧。
  • Expresso 是一个更传统的单元测试框架。
  • 节点单元是另一个相对传统的单元测试框架。
  • AVA 是一个新的测试运行者，内置Babel并同时运行测试。

网页框架

  • Express.js 是目前最流行的框架。
  • Koa 是由Express.js团队设计的一个新的Web框架，旨在成为一个更小，更为Web应用程序和API提供更具表现力，更强大的基础。
  • sails.js 是最流行的Node.js MVC框架，基于express。它旨在模拟熟悉的Ruby on Rails框架MVC模式，但支持现代应用程序的要求：数据驱动的API与可扩展的面向服务的体系结构。
  • Meteor 将jQuery，Handlebars，Node.js， WebSocket ， MongoDB 和DDP，并且不使用 Ruby on Rails 克隆来提升约定而不是配置。
  • Tower （ deprecated ）是Express.js的顶层抽象，旨在成为Ruby on Rails克隆。
  • Geddy 是Web框架的另一个亮点。
  • RailwayJS 是一个Ruby on Rails灵感的MVC web框架。
  • Sleek.js 是一个基于Express.js构建的简单Web框架。
  • Hapi 是一个以配置为中心的框架，内置了对输入验证，缓存，认证等的支持。 LI>

  • Trails 是一个现代化的Web应用程序框架。它建立在 Rails 和 Grails ，通过坚持直接的，基于约定的API驱动设计理念来加速开发。

  • Danf 是一个完整的OOP框架，提供了许多功能，以便生成一个可扩展，可维护，可测试和高性能的应用程序，并允许在服务器（Node.js）和客户端（浏览器）两端以相同的方式进行编码。

  • Derbyjs 是一个反应式的全栈JavaScript框架。他们正在使用像反应式编程和同构JavaScript的模式很长一段时间。

  • Loopback.io 是一个功能强大的Node.js框架，用于创建API并轻松连接到后端数据源。它有一个Angular.js SDK，为iOS和Android提供SDK。

Web框架工具：

  • Jade 是Node.js世界的HAML / Slim
  • EJS 是一种更传统的模板语言。
  • 不要忘记下划线的模板方法！

<强>网络：

  • 连接是Node.js世界的Rack或WSGI。
  • 请求是非常受欢迎的HTTP请求库。
  • socket.io 便于构建WebSocket服务器。

命令行交互：

  • minimist 只是命令行参数解析。
  • Yargs 是一个用于解析命令行参数的功能强大的库。
  • Commander.js 是用于构建一次性命令行应用程序的完整解决方案。< / LI>
  • Vorpal.js 是一个构建成熟的沉浸式命令行应用程序的框架。
  • 粉笔使您的CLI输出变得非常漂亮。
