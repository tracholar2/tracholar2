你如何检查JavaScript中的空字符串？

By zhizi

Published 2018-01-14

Contents

我看到这个线程，但我没有看到一个JavaScript的具体例子。在JavaScript中是否有一个简单的string.Empty，或者只是检查“？` 如果你只是想检查是否有任何价值，你可以做

if (strValue) {
    //do something
}

如果您需要特别检查空字符串是否超过null，我认为检查“是最好的选择，使用 ===运算符（这样你就知道它实际上是一个你要比较的字符串）。

为了检查一个字符串是否为空，空或未定义，我使用：

function isEmpty(str) {
    return (!str || 0 === str.length);
}

为了检查一个字符串是否为空，null或undefined，我使用：

function isBlank(str) {
    return (!str || /^\s*$/.test(str));
}

用于检查一个字符串是空白还是只包含空格

String.prototype.isEmpty = function() {
    return (this.length === 0 || !this.trim());
};
