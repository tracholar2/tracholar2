如何删除选择框的所有选项，然后添加一个选项，并选择与jQuery？

使用核心jQuery，如何删除选择框的所有选项，然后添加一个选项并选择它？

我的选择框如下。

<Select id="mySelect" size="9" </Select>

编辑：下面的代码是有用的链接。但是，（在Internet Explorer中）.val（'whatever'）没有选择添加的选项。 （我在.append和.val中使用了相同的’value’）。

$('#mySelect').find('option').remove().end().append('<option value="whatever">text</option>').val('whatever');

编辑：试图让它模仿这个代码，每当页面/窗体重置时，我使用下面的代码。这个选择框由一组单选按钮组成。 .focus（）更接近，但该选项没有像.selected =“true”那样选择。没有什么是我现有的代码错了 - 我只是想学习jQuery。

var mySelect = document.getElementById('mySelect');
mySelect.options.length = 0;
mySelect.options[0] = new Option ("Foo (only choice)", "Foo");
mySelect.options[0].selected="true";

编辑：选择的答案是接近我所需要的。这对我有用：

$('#mySelect').children().remove().end().append('<option selected value="whatever">text</option>') ;

但是两个答案都让我得到了最终的解决方案。

$('#mySelect')
    .find('option')
    .remove()
    .end()
    .append('<option value="whatever">text</option>')
    .val('whatever')
;



$('#mySelect')
    .empty()
    .append('<option selected="selected" value="whatever">text</option>')
;
