如何使用jQuery更改超链接的href？

$("a").attr("href", "http://www.google.com/")

…将所有超链接的href修改为指向Google。你可能想要一个更精致的选择器。例如，如果你有链接源（超链接）和链接目标（a.k.a.“anchor”）锚定标记的混合：

<a name="MyLinks"></a>
<a href="http://www.codeproject.com/>The CodeProject</a>

…那么你可能不希望为他们添加href属性。为了安全起见，我们可以指定我们的选择器只会将＆lt; a＆gt;标签与现有的href属性匹配：

$("a[href]") //...

当然，你可能会想一些更有趣的事情。如果你想匹配一个特定的现有href的锚，你可以使用这样的东西：

$("a[href='http://www.google.com/']").attr('href', 'http://www.live.com/')

这将找到href完全匹配字符串http://www.google.com/的链接。一个更复杂的任务可能是匹配，然后只更新href的一部分：

$("a[href^='http://stackoverflow.com']")
   .each(function()
   { 
      this.href = this.href.replace(/^http:\/\/beta\.stackoverflow\.com/, 
         "http://stackoverflow.com");
   });

第一部分只用http://stackoverflow.com选择href 开始
的链接。然后定义一个函数，使用一个简单的正则表达式来替换这部分URL。请注意这种灵活性 - 链接的任何修改都可以在这里完成。

使用jQuery 1.6及以上版本，您应该使用：

$("a").prop("href", "http://www.jakcms.com")

prop和attr之间的区别在于attr抓住了prop属性获取DOM属性的HTML属性。
