检查一个变量是否是一个整数

By zhizi

Published 2018-01-14

Contents
  1. 1. BUT

如何检查一个变量是否是一个整数？

如果您需要这样做，请执行

isinstance(<var>, int)

除非你在Python 2.x中，在这种情况下你想要

isinstance(<var>, (int, long))

不要使用type。 Python几乎从来都不是正确的答案，因为它阻止了多态的所有灵活性。例如，如果你继承int，你的新类应该注册为int，type不会这样做：

class Spam(int): pass
x = Spam(0)
type(x) == int # False
isinstance(x, int) # True

这符合Python强大的多态性：你应该允许任何象int那样的对象，而不是强制它成为一个对象。

BUT

然而，经典的Python思路是，请求宽恕比允许更容易_。换句话说，不要检查x是否是一个整数;假设它是，并且如果不是，则捕获异常结果：

try:
    x += 1
except TypeError:
    ...

这种心态正在慢慢被使用抽象基类所取代，它们让你通过从一个专门构造的类继承来准确地注册你的对象应该具有的属性（添加？倍增？）。这将是最好的解决方案，因为它可以准确地_那些具有必要和足够属性的对象，但是你必须阅读关于如何使用它的文档。

到目前为止，所有提出的答案都似乎错过了一个事实，即双精度浮点数（实际上是双精度浮点数）也可以是一个整数（如果小数点后面没有任何内容）。我使用内置的is_integer（）方法来检查这个。

举例（在for循环中每隔一段时间做一些事情）：

for index in range(y): 
    # do something
    if (index/x.).is_integer():
        # do something special

编辑：

在调用这个方法之前，你总是可以转换成一个浮点数。三种可能性：

>>> float(5).is_integer()
True
>>> float(5.1).is_integer()
False
>>> float(5.0).is_integer()
True

否则，你可以像Agostino那样检查它是否是一个int：

def is_int(val):
    if type(val) == int:
        return True
    else:
        if val.is_integer():
            return True
        else:
            return False
