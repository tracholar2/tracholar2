Type checking

使用抽象基础类。他们至少需要Python
2.6，并且只能用于新风格的类。

import collections

if isinstance(theElement, collections.Iterable):
    # iterable
else:
    # not iterable

然而，collections.Iterable不会检查__ getitem __，所以iter更可靠。
