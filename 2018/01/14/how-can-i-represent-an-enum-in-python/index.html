我怎样才能在Python中表示一个'Enum'？

By zhizi

Published 2018-01-14

Contents

我主要是一名C＃开发人员，但是我目前正在使用Python开发一个项目。

我怎样才能代表Python中的Enum等价物？

如 PEP 435所述，Enum已经被添加到Python 3.4中。它也被backported到3.3,3.2,3.1,2.7,2.6,2.5和2.4的PyPI。

对于更高级的枚举技术，请尝试使用aenum库（2.7,3.3+，与author相同代码在py2和py3之间并不完全兼容，例如你需要__ order __ code> in python 2）。

  • 要使用enum34，请执行$ pip install enum34 * 要使用aenum，请执行$ pip install aenum
    安装enum（没有数字）会安装一个完全不同的，不兼容的版本。
from enum import Enum     # for enum34, or the stdlib version
# from aenum import Enum  # for the aenum version
Animal = Enum('Animal', 'ant bee cat dog')

Animal.ant  # returns <Animal.ant: 1>
Animal['ant']  # returns <Animal.ant: 1> (string lookup)
Animal.ant.name  # returns 'ant' (inverse lookup)

或等同：

class Animal(Enum):
    ant = 1
    bee = 2
    cat = 3
    dog = 4

在早期版本中，完成枚举的一种方法是：

def enum(**enums):
    return type('Enum', (), enums)

这样使用：

>>> Numbers = enum(ONE=1, TWO=2, THREE='three')
>>> Numbers.ONE
1
>>> Numbers.TWO
2
>>> Numbers.THREE
'three'

你也可以很容易地支持像这样的自动枚举：

def enum(*sequential, **named):
    enums = dict(zip(sequential, range(len(sequential))), **named)
    return type('Enum', (), enums)

并像这样使用：

>>> Numbers = enum('ZERO', 'ONE', 'TWO')
>>> Numbers.ZERO
0
>>> Numbers.ONE
1

支持将值转换回名称可以这样添加：

def enum(*sequential, **named):
    enums = dict(zip(sequential, range(len(sequential))), **named)
    reverse = dict((value, key) for key, value in enums.iteritems())
    enums['reverse_mapping'] = reverse
    return type('Enum', (), enums)

这会覆盖这个名字的任何东西，但是在输出中渲染你的枚举是很有用的。如果反向映射不存在，它将抛出KeyError。第一个例子：

>>> Numbers.reverse_mapping['three']
'THREE'

在PEP 435之前，Python没有相同的功能，但是你可以实现自己的功能。

我自己，我喜欢保持简单（我已经看到网络上一些可怕的复杂的例子），像这样的…

class Animal:
    DOG = 1
    CAT = 2

x = Animal.DOG

在Python 3.4（ PEP 435）中，你可以使Enum基类。这可以让你获得一些额外的功能，在PEP中描述。例如，枚举值不同于整数。

class Animal(Enum):
    DOG = 1
    CAT = 2

print(Animal.DOG)
<Animal.DOG: 1>

如果您不想输入值，请使用以下快捷键：

class Animal(Enum):
    DOG, CAT = range(2)
