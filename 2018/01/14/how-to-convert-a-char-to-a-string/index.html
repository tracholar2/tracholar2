如何将字符转换为字符串？

By zhizi

Published 2018-01-14

Contents

我有一个char，我需要一个String。如何从一个转换到另一个？

您可以使用 ` Character.toString（char）的。请注意，此方法只是将调用返回给 String.valueOf（char），这也是可行的。

正如其他人所指出的，字符串连接也是一个捷径：

String s = "" + 's';

但是这个编译到：

String s = new StringBuilder().append("").append('s').toString();

效率较低，因为 StringBuilder 由 char [] 支持（由[ StringBuilder（） ](“http://grepcode.com/file)转换为 16 ），仅用于由 String 进行防御复制的数组。String.valueOf（char）通过将 char 封装在一个单一元素数组中，并将其传递给包的私有构造函数[ String（char
]，boolean）`，避免数组拷贝。

很好的问题。我有以下五种方法来做到这一点。

1. String stringValueOf = String.valueOf('c'); // most efficient

2. String stringValueOfCharArray = String.valueOf(new char[]{x});

3. String characterToString = Character.toString('c');

4. String characterObjectToString = new Character('c').toString();

   // Although this method seems very simple, 
   // this is less efficient because the concatenation
   // expands to new StringBuilder().append(x).append("").toString();
5. String concatBlankString = 'c' + "";

6. String fromCharArray = new String(new char[]{x});

Note:
Character.toString(char)
returns
String.valueOf(char).
So effectively both are same.

String.valueOf（char [] value）调用新的String（char [] value），它依次设置值 char阵列。

public String(char value[]) {
    this.value = Arrays.copyOf(value, value.length);
}

另一方面，String.valueOf（char value）调用下面的 包私有构造函数。

String(char[] value, boolean share) {
    // assert share : "unshared not supported";
    this.value = value;
}
