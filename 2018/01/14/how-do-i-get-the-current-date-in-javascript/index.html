我如何在JavaScript中获取当前日期？

使用new Date（）生成包含当前日期和时间的新的Date对象。

var today = new Date();
var dd = today.getDate();
var mm = today.getMonth()+1; //January is 0!
var yyyy = today.getFullYear();

if(dd<10) {
    dd = '0'+dd
} 

if(mm<10) {
    mm = '0'+mm
} 

today = mm + '/' + dd + '/' + yyyy;
document.write(today);

这会以mm / dd / yyyy的格式显示今天的日期。

只要将today = mm +'/'+ dd +'/'+ yyyy;更改为您希望的任何格式即可。

var utc = new Date().toJSON().slice(0,10).replace(/-/g,'/');
document.write(utc);

如果要重复使用utc变量（如new Date（utc））作为Firefox，请使用replace选项。
Safari无法识别破折号的日期。
