如何在Python中获取文件创建和修改日期/时间？

By zhizi

Published 2018-01-14

Contents

我有一个脚本，需要做一些基于文件创建＆amp;修改日期，但必须在Linux上运行。视窗。

什么是最好的 跨平台 的方式来获得文件创建＆amp;修改Python中的日期/时间？

以跨平台方式获取修改日期非常简单 - 只需调用 os.path.getmtime（ _），你将得到path代码>最后修改。

另一方面，获取文件 创建 日期依赖于平台和平台，甚至在三个大操作系统之间也是不同的：

  • 在 Windows 上，文件的ctime（记录在 https://msdn.microsoft.com/en-us/library/14h5k7ff.aspx ）存储其创建日期。您可以通过 os.path.getctime （） 或 .st_ctime 属性的调用结果 <代码> os.stat（） 。这个 不会 在Unix上工作，其中ctime 是文件属性 或 内容被更改的最后一次。
  • 在 Mac 以及其他一些基于Unix的操作系统上，您可以使用 os.stat（）的结果的属性“os.stat_result.st_birthtime”rel =“noreferrer”> .st_birthtime ` 在 Linux 上，目前这是不可能的，至少不用为Python编写C扩展。虽然一些通常用于Linux的文件系统可以创建存储日期（例如，ext4将它们存储在st_crtime中），Linux内核提供无法访问它们的方法;特别是从C中调用的stat（）返回的结构，就像最新的内核版本不包含任何创建日期字段。您还可以看到st_crtime标识符目前不在 Python源代码。至少在ext4中，数据 _被附加到文件系统中的inode，但没有方便的方式访问它。

Linux上最好的事情是访问文件的mtime，通过结果的os.html＃os.stat_result.st_ctime“rel
=”noreferrer“> .st_mtime属性。这会给你最后一次文件内容被修改，这可能足以满足一些使用情况。

把这一切放在一起，跨平台的代码应该看起来像这样…

import os
import platform

def creation_date(path_to_file):
    """
    Try to get the date that a file was created, falling back to when it was
    last modified if that isn't possible.
    See http://stackoverflow.com/a/39501288/1709587 for explanation.
    """
    if platform.system() == 'Windows':
        return os.path.getctime(path_to_file)
    else:
        stat = os.stat(path_to_file)
        try:
            return stat.st_birthtime
        except AttributeError:
            # We're probably on Linux. No easy way to get creation dates here,
            # so we'll settle for when its content was last modified.
            return stat.st_mtime

你有几个选择。首先，您可以使用 os.path.getmtime和 os.path .getctime功能：

import os.path, time
print("last modified: %s" % time.ctime(os.path.getmtime(file)))
print("created: %s" % time.ctime(os.path.getctime(file)))

您的其他选项是使用 os.stat ：

import os, time
(mode, ino, dev, nlink, uid, gid, size, atime, mtime, ctime) = os.stat(file)
print("last modified: %s" % time.ctime(mtime))
