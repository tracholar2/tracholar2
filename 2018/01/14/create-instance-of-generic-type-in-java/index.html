在Java中创建泛型类型的实例？

By zhizi

Published 2018-01-14

Contents

是否有可能在Java中创建一个泛型类型的实例？我基于我所看到的答案是no（ 由于类型擦除
），但我会感兴趣，如果任何人都可以看到我的东西，我错过了：

class SomeContainer<E>
{
    E createContents()
    {
        return what???
    }
}

编辑：事实证明，可以使用超级类型令牌为了解决我的问题，但它需要大量的基于反射的代码，如下面的一些答案已经表明。

我会把这个开放一会儿，看看有没有什么比Ian Robertson的 Artima Article。

你是对的你不能做“新E”。但是你可以把它改成

private static class SomeContainer<E>
{
    E createContents(Class<E> clazz)
    {
        return clazz.newInstance();
    }
}

这是一个痛苦。但它的作品。用工厂模式包装它使得它更容易一些。

不知道如果这有帮助，但是当你的子类（包括匿名）泛型，类型信息可以通过反射。如，

public abstract class Foo<E> {

  public E instance;  

  public Foo() throws Exception {
    instance = ((Class)((ParameterizedType)this.getClass().
       getGenericSuperclass()).getActualTypeArguments()[0]).newInstance();
    ...
  }

}

所以，当你继承Foo的时候，你会得到一个Bar的实例，例如

// notice that this in anonymous subclass of Foo
assert( new Foo<Bar>() {}.instance instanceof Bar );

但是这是很多的工作，只适用于子类。可以很方便。
