将字符串转换为日期时间

By zhizi

Published 2018-01-14

Contents

简单而简单。我有一个像这样的日期时间的巨大列表：

Jun 1 2005  1:33PM
Aug 28 1999 12:00AM

我将把它们推回到数据库中正确的日期时间字段，所以我需要把它们变成真正的日期时间对象。

任何帮助（即使这只是一个正确的方向踢），将不胜感激。

编辑：这是通过Django的ORM，所以我不能使用SQL做插入转换。

from datetime import datetime

datetime_object = datetime.strptime('Jun 1 2005  1:33PM', '%b %d %Y %I:%M%p')

产生的datetime对象是没有时区的。

链接：

strptime的Python文档： Python 2，Python 3

strftime格式的Python文档掩码： Python 2， Python 3

  • strftime.org 也是strftime的一个很好的参考

注意：

  • strptime =“字符串解析时间”
  • strftime =“字符串格式化时间”
  • 今天大声地发出声音＆amp;您将不必在6个月内再次搜索。

使用第三方 dateutil 图书馆：

from dateutil import parser
dt = parser.parse("Aug 28 1999 12:00AM")

它可以处理大多数的日期格式，包括你需要解析的格式。它比strptime更方便，因为它大部分时间都能猜出正确的格式。

编写测试非常有用，可读性比性能更重要。

pip install python-dateutil
