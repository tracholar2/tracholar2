用JavaScript比较两个日期

有人可以提供一种方法来比较 两个日期 的值大于，小于，而不是过去使用JavaScript？这些值将来自文本框…

日期对象将执行您需要 -
为每个日期构建一个，然后使用＆gt;，＆lt; / code>，＆lt; = 或＆gt; = 。

==，！=，===和！==运算符要求您在

var d1 = new Date();
var d2 = new Date(d1);
var same = d1.getTime() === d2.getTime();

明白只是直接检查与数据对象的相等是行不通的

var d1 = new Date();
var d2 = new Date(d1);

console.log(d1 == d2);   // prints false (wrong!) 
console.log(d1 === d2);  // prints false (wrong!)
console.log(d1 != d2);   // prints true  (wrong!)
console.log(d1 !== d2);  // prints true  (wrong!)
console.log(d1.getTime() === d2.getTime()); // prints true (correct)

我建议你使用下拉式或类似的日期输入约束形式，而不是文本框，以免发现自己在输入验证地狱中。

在javascript中比较日期的最简单方法是首先将其转换为Date对象，然后比较这些日期对象。

在下面你找到一个有三个功能的对象：

  • <强> dates.compare（A，B）

返回一个数字：

* -1如果a < B'/ LI> 
* 0如果a = b 
* 1如果a＆gt; B'/ LI> 
* 如果a或b是非法日期，则为NaN
  • dates.inRange （d，开始，结束）

返回一个布尔值或NaN：

如果 d 位于 start 和 end （包含）之间， _ 如果 d 在 _start __或 end 之后_，

* _
* 如果有一个或多个日期是非法的，则为NaN。
  • <强> dates.convert

被其他函数用来将它们的输入转换为一个日期对象。输入可以是

* a **日期** -object：输入按原样返回。
* **数组** ：解释为[年，月，日]。 **注意** 月份是0-11。
* **数字** ：解释为自1970年1月1日以来的毫秒数（时间戳）
* a **字符串** ：支持多种不同的格式，例如“YYYY / MM / DD”，“MM / DD / YYYY”，“ 
* **对象** ：被解释为具有年份，月份和日期属性的对象。 **注意** 月份是0-11。



// Source: http://stackoverflow.com/questions/497790
var dates = {
    convert:function(d) {
        // Converts the date in d to a date-object. The input can be:
        //   a date object: returned without modification
        //  an array      : Interpreted as [year,month,day]. NOTE: month is 0-11.
        //   a number     : Interpreted as number of milliseconds
        //                  since 1 Jan 1970 (a timestamp) 
        //   a string     : Any format supported by the javascript engine, like
        //                  "YYYY/MM/DD", "MM/DD/YYYY", "Jan 31 2009" etc.
        //  an object     : Interpreted as an object with year, month and date
        //                  attributes.  **NOTE** month is 0-11.
        return (
            d.constructor === Date ? d :
            d.constructor === Array ? new Date(d[0],d[1],d[2]) :
            d.constructor === Number ? new Date(d) :
            d.constructor === String ? new Date(d) :
            typeof d === "object" ? new Date(d.year,d.month,d.date) :
            NaN
        );
    },
    compare:function(a,b) {
        // Compare two dates (could be of any type supported by the convert
        // function above) and returns:
        //  -1 : if a < b
        //   0 : if a = b
        //   1 : if a > b
        // NaN : if a or b is an illegal date
        // NOTE: The code inside isFinite does an assignment (=).
        return (
            isFinite(a=this.convert(a).valueOf()) &&
            isFinite(b=this.convert(b).valueOf()) ?
            (a>b)-(a<b) :
            NaN
        );
    },
    inRange:function(d,start,end) {
        // Checks if date in d is between dates in start and end.
        // Returns a boolean or NaN:
        //    true  : if d is between start and end (inclusive)
        //    false : if d is before start or after end
        //    NaN   : if one or more of the dates is illegal.
        // NOTE: The code inside isFinite does an assignment (=).
       return (
            isFinite(d=this.convert(d).valueOf()) &&
            isFinite(start=this.convert(start).valueOf()) &&
            isFinite(end=this.convert(end).valueOf()) ?
            start <= d && d <= end :
            NaN
        );
    }
}
