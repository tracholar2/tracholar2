如何知道一个对象是否在Python中有一个属性

By zhizi

Published 2018-01-14

Contents

Python有没有一种方法来确定一个对象是否有一些属性？例如：

>>> a = SomeClass()
>>> a.someProperty = value
>>> a.property
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
AttributeError: SomeClass instance has no attribute 'property'

在使用它之前，如何知道a是否具有属性property？

尝试hasattr（）：

if hasattr(a, 'property'):
    a.property

编辑：请参阅下面的zweiterlinde的回答，他提供了关于请求宽恕的好建议！非常pythonic方法！

python的一般做法是，如果属性可能在大多数情况下存在，只需调用它，或者让异常传播，或者用try / except块来捕获它。这可能会比hasattr更快。如果这个属性可能在大多数情况下不存在，或者你不确定，那么使用hasattr可能会比重复进入一个异常块更快。

由于 Jarret Hardie 回答，hasattr会做的。然而，我想补充一点，Python社区中的许多人推荐了一种“比请求宽容而不是许可”（EAFP）的策略，而不是“先看你跳跃”（LBYL）。请参阅这些参考资料：

EAFP vs
LBYL（回复：目前有点失望）
EAFP vs. LBYL @Code就像一个Pythonista：习惯Python

即：

try:
    doStuff(a.property)
except AttributeError:
    otherStuff()

…优先于：

if hasattr(a, 'property'):
    doStuff(a.property)
else:
    otherStuff()
