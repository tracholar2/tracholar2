使用“for”循环遍历字典

By zhizi

Published 2018-01-14

Contents

我对以下代码感到有点困惑：

d = {'x': 1, 'y': 2, 'z': 3} 
for key in d:
    print key, 'corresponds to', d[key]

我不明白的是键部分。 Python如何识别它只需要从字典中读取密钥？ key在Python中是一个特殊的词吗？或者它只是一个变量？

key `只是一个变量名。

for key in d:

将简单地遍历字典中的键，而不是键和值。要循环使用键和值，可以使用以下命令：

对于Python 2.x：

for key, value in d.iteritems():

对于Python 3.x：

for key, value in d.items():

要自己测试，请将键改为poop。

对于Python 3.x，iteritems（）已经被简单的items（）所取代，它返回一个类似于dict的类似视图，比如iteritems（），但更好。 这也可以在2.7中作为viewitems（）使用。

操作items（）对于2和3都是适用的，但是在2中它将返回字典的（key，value）对的列表，这将不会反映在items（）调用之后发生的字典变化。如果你想在3.x中使用2.x行为，你可以调用list（d.items（））。

这不是关键是一个特殊的词，但字典实现迭代器协议。你可以在课堂上做到这一点，例如请参阅这个问题了解如何构建类迭代器。

在字典的情况下，它在C级执行。详情请参阅 PEP 234。特别是“词典迭代器”部分：

  • Dictionaries implement a tp_iter slot that returns an efficient iterator
    that iterates over the keys of the dictionary. […] This means that we can
    write
>     for k in dict: ...

which is equivalent to, but much faster than

>     for k in dict.keys(): ...

as long as the restriction on modifications to the dictionary (either by the
loop or by another thread) are not violated.

>

  • Add methods to dictionaries that return different kinds of iterators
    explicitly:
>     for key in dict.iterkeys(): ...


for value in dict.itervalues(): ...


for key, value in dict.iteritems(): ...

This means that for x in dict is shorthand for for x in dict.iterkeys().

>
