如何在Python中解析XML？

By zhizi

Published 2018-01-14

Contents

在包含xml的数据库中有很多行，我试图编写一个Python脚本来检查这些行，并计算特定节点属性的实例数。例如，我的树看起来像：

<foo>
   <bar>
      <type foobar="1"/>
      <type foobar="2"/>
   </bar>
</foo>

如何使用Python访问XML中的属性1和2？

我建议 ElementTree。还有其他兼容的API实现，比如lxml 和cElementTree在Python标准库本身;但是在这种情况下，他们主要添加的是更快的速度 - 编程部分的简单性取决于API，ElementTree定义了`。

在从XML构建元素实例e之后，例如使用 XML函数，或通过用类似

的方式解析文件

import xml.etree.ElementTree
e = xml.etree.ElementTree.parse('thefile.xml').getroot()

或 ElementTree中显示的许多其他方法>，你只要做一些事情：

for atype in e.findall('type'):
    print(atype.get('foobar'))

以及类似的，通常很简单的代码模式。

minidom是最快最直截了当：

XML：

<data>
    <items>
        <item name="item1"></item>
        <item name="item2"></item>
        <item name="item3"></item>
        <item name="item4"></item>
    </items>
</data>

PYTHON：

from xml.dom import minidom
xmldoc = minidom.parse('items.xml')
itemlist = xmldoc.getElementsByTagName('item')
print(len(itemlist))
print(itemlist[0].attributes['name'].value)
for s in itemlist:
    print(s.attributes['name'].value)

输出

4
item1
item1
item2
item3
item4
