用JavaScript获取当前的URL？

By zhizi

Published 2018-01-14

Contents

我只想得到网站的网址。不是从链接中获取的网址。在页面加载，我需要能够抓住网站的完整，当前的网址，并将其设置为一个变量做我喜欢。

使用：

window.location.href

正如在评论中指出的那样，下面这行代码行得通，但是对于Firefox来说是错误的。

document.URL;

请参阅 DOMString类型的URL，只读 。

JavaScript提供了许多方法来检索和更改浏览器地址栏中显示的当前URL。所有这些方法都使用Location对象，它是Window对象的一个​​属性。您可以创建一个具有当前URL的新的Location对象，如下所示：

var currentLocation = window.location;

网址的基本结构

<protocol>//<hostname>:<port>/<pathname><search><hash>

协议 - 指定用于访问Internet上资源的协议名称。 （HTTP（不带SSL）或HTTPS（带SSL））

  • hostname - 主机名称指定拥有资源的主机。例如，www.stackoverflow.com。服务器使用主机名称提供服务。
    端口号 - 用于识别互联网或其他网络消息到达服务器时要转发的特定进程的端口号。

  • 路径名 - 该路径提供有关Web客户端要访问的主机内的特定资源的信息。例如，/index.html。

查询 - 查询字符串遵循路径组件，并提供资源可用于某种目的（例如，作为搜索参数或待处理数据）的信息的字符串。

  • 散列 - URL的锚点部分，包含散列符号（＃）。

使用这些Location对象属性，您可以访问所有这些URL组件

  • 散列 - 设置或返回URL的锚点部分。
  • 主机 - 设置或返回URL的主机名和端口。
  • 主机名 - 设置或返回URL的主机名。
  • href - 设置或返回整个网址。
  • 路径名 - 设置或返回URL的路径名称。
  • port - 设置或返回服务器用于URL的端口号。
  • 协议 - 设置或返回URL的协议。
  • 搜索 - 设置或返回URL的查询部分
