如何按价值排序字典？

By zhizi

Published 2018-01-14

Contents
  1. 1. As simple as: sorted(dict1, key=dict1.get)

我有一个从数据库中的两个字段读取的值的字典：一个字符串字段和一个数字字段。字符串字段是唯一的，所以这是字典的关键。

我可以按键排序，但是如何根据值排序？

注意：我已经阅读了堆栈溢出问题 _[如何根据Python中字典的值对字典列表进行排序？

a> _ ，并且可能会将我的代码更改为拥有一个字典列表，但是因为我不需要一个字典列表，所以我想知道是否有更简单的解决方案。

不能排序字典，只能得到一个已排序字典的表示形式。字典本质上是无序的，但其他类型，如列表和元组，不是。所以你需要一个有序的数据类型来表示排序的值，这将是一个列表

  • 可能是一个元组列表。

例如，

import operator
x = {1: 2, 3: 4, 4: 3, 2: 1, 0: 0}
sorted_x = sorted(x.items(), key=operator.itemgetter(1))

sorted_x 将是每个元组中第二个元素排序的元组列表。 dict（sorted_x）== x `。

对于那些希望按键而不是数值的人来说：

import operator
x = {1: 2, 3: 4, 4: 3, 2: 1, 0: 0}
sorted_x = sorted(x.items(), key=operator.itemgetter(0))

As simple as: sorted(dict1, key=dict1.get)

那么，实际上可以做一个“按字典值排序”。最近我不得不在代码高尔夫球（代码高尔夫球）中做到这一点（堆栈溢出问题
_代码高尔夫球：词频图）。摘要的问题是这样的：给定一个文本，计算每个单词遇到的频率，并显示一个顶级单词列表，按频率降序排列。

如果你构造一个字典作为关键字和每个字的出现次数作为值，在这里简化为：

from collections import defaultdict
d = defaultdict(int)
for w in text.split():
  d[w] += 1

然后你可以得到一个单词列表，按照使用频率排序，用sorted（d，key = d.get） - 排序迭代字典键，使用单词数作为排序键发生。

for w in sorted(d, key=d.get, reverse=True):
  print w, d[w]
