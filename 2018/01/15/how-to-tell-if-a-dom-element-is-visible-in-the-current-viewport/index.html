如何判断一个DOM元素在当前视口中是否可见？

By zhizi

Published 2018-01-15

Contents

有没有一种有效的方式来判断一个DOM元素（在HTML文档中）当前是否可见（出现在 视口 中）？

（关于Firefox的问题）

更新： 时间流逝，浏览器也如此。 不再推荐此技术 ，您应该在下面使用@ Dan的解决方案（https://stackoverflow.com/ a / 7557433/5628）如果你不需要支持IE＆lt; 7。

原始解决方案（现已过时）：**

这将检查元素在当前视口中是否完全可见：

function elementInViewport(el) {
  var top = el.offsetTop;
  var left = el.offsetLeft;
  var width = el.offsetWidth;
  var height = el.offsetHeight;

  while(el.offsetParent) {
    el = el.offsetParent;
    top += el.offsetTop;
    left += el.offsetLeft;
  }

  return (
    top >= window.pageYOffset &&
    left >= window.pageXOffset &&
    (top + height) <= (window.pageYOffset + window.innerHeight) &&
    (left + width) <= (window.pageXOffset + window.innerWidth)
  );
}

您可以简单地修改这个元素，以确定元素的任何部分在视口中是否可见：

function elementInViewport2(el) {
  var top = el.offsetTop;
  var left = el.offsetLeft;
  var width = el.offsetWidth;
  var height = el.offsetHeight;

  while(el.offsetParent) {
    el = el.offsetParent;
    top += el.offsetTop;
    left += el.offsetLeft;
  }

  return (
    top < (window.pageYOffset + window.innerHeight) &&
    left < (window.pageXOffset + window.innerWidth) &&
    (top + height) > window.pageYOffset &&
    (left + width) > window.pageXOffset
  );
}

现在大多数浏览器都支持getBoundingClientRect方法，这已经成为最佳实践。使用旧的答案是很慢，不准确和有几个错误。

选择正确的解决方案是几乎从不精确。您可以阅读更多关于它的错误。

这个解决方案已经在IE7 +，iOS5 + Safari，Android2 +，Blackberry，Opera Mobile和IE
Mobile上进行了测试 10 

function isElementInViewport (el) {

    //special bonus for those using jQuery
    if (typeof jQuery === "function" && el instanceof jQuery) {
        el = el[0];
    }

    var rect = el.getBoundingClientRect();

    return (
        rect.top >= 0 &&
        rect.left >= 0 &&
        rect.bottom <= (window.innerHeight || document.documentElement.clientHeight) && /*or $(window).height() */
        rect.right <= (window.innerWidth || document.documentElement.clientWidth) /*or $(window).width() */
    );
}

How to use:

你可以确定上面给出的函数在被调用的时候返回正确的答案，但是跟踪元素的可见性作为一个事件呢？

将下面的代码放在＆lt; body＆gt;标记的底部：

function onVisibilityChange(el, callback) {
    var old_visible;
    return function () {
        var visible = isElementInViewport(el);
        if (visible != old_visible) {
            old_visible = visible;
            if (typeof callback == 'function') {
                callback();
            }
        }
    }
}

var handler = onVisibilityChange(el, function() {
    /* your code go here */
});


//jQuery
$(window).on('DOMContentLoaded load resize scroll', handler);
