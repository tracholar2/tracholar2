如何使用可点击的标签创建HTML复选框

By zhizi

Published 2018-01-15

Contents
  1. 1. Method 1: Wrap Label Tag
  2. 2. Method 2: Use the for Attribute
  3. 3. Explanation

如何创建一个可点击标签的HTML复选框（这意味着点击标签会打开/关闭复选框）？

Method 1: Wrap Label Tag

将复选框包装在标签标签内：

<label><input type="checkbox" name="checkbox" value="value">Text</label>

Method 2: Use the for Attribute

使用for属性（匹配复选框id）：

<input type="checkbox" name="checkbox" id="checkbox_id" value="value">
<label for="checkbox_id">Text</label>

注意 ：网页上的ID必须是唯一的！

Explanation

由于其他答案没有提及它，因此标签最多可包含1个输入，并省略属性的，并假定它是用于输入的。 p> 摘自 w3.org（与我的重点）：

[The for attribute] explicitly associates the label being defined with
another control. When present, the value of this attribute must be the same as
the value of the id attribute of some other control in the same document.
When absent, the label being defined is associated with the element’s
contents.

>

To associate a label with another control implicitly, the control element
must be within the contents of the LABEL element. In this case, the LABEL
may only contain one control element. The label itself may be positioned
before or after the associated control.

使用这个方法比有一些优点：

  • 不需要为每个复选框分配一个id（很棒！）。

  • 不需要在＆lt; label＆gt;中使用额外的属性。

  • 输入的可点击区域也是标签的可点击区域，所以没有两个单独的地方可以控制复选框 - 只有一个，无论距离`＆lt; input＆gt; < / code>和实际的标签文本，无论你使用什么样的CSS。

用一些CSS演示：

label {
 border:1px solid #ccc;
 padding:10px;
 margin:0 0 10px;
 display:block; 
}

label:hover {
 background:#eee;
 cursor:pointer;
}


<label><input type="checkbox" />Option 1</label>
<label><input type="checkbox" />Option 2</label>
<label><input type="checkbox" />Option 3</label>

只要确保标签与输入相关联即可。

<fieldset>
  <legend>What metasyntactic variables do you like?</legend>

  <input type="checkbox" name="foo" value="bar" id="foo_bar">
  <label for="foo_bar">Bar</label>

  <input type="checkbox" name="foo" value="baz" id="foo_baz">
  <label for="foo_baz">Baz</label>
</fieldset>
