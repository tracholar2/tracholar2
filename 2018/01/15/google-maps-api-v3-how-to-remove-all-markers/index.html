map.clearOverlays();

只需执行以下操作：

予。声明一个全局变量：

var markersArray = [];

II。定义一个函数：

function clearOverlays() {
  for (var i = 0; i < markersArray.length; i++ ) {
    markersArray[i].setMap(null);
  }
  markersArray.length = 0;
}

OR

google.maps.Map.prototype.clearOverlays = function() {
