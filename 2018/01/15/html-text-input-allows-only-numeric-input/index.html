HTML文本输入只允许数字输入

By zhizi

Published 2018-01-15

Contents

有没有一种快速的方法来设置一个HTML文本输入（＆lt; input type = text /＆gt;）只允许数字键击（加’。’）？

<强>的JavaScript （最可靠的）

<input type="text" onkeypress='return event.charCode >= 48 && event.charCode <= 57'>
</input>

虽然这很简单，但它不会让你使用组合键和其他不可键入的键。对于一个更完整的JavaScript解决方案，也支持类型号码和最大长度验证的输入，可以考虑使用这个Polyfill

HTML 5 与＆lt;！DOCTYPE html＆gt;有原生解决方案：

＆lt; input type =“number”＆gt;
请注意，在某些浏览器中的行为不符合标准。

尝试使用输入类型=数字，以查看HTML5版本的实际应用

另请参阅 https://github.com/jonstipe/number-polyfill获取透明支持在较旧的浏览器中。

<强>的jQuery

$(document).ready(function() {
    $("#txtboxToFilter").keydown(function (e) {
        // Allow: backspace, delete, tab, escape, enter and .
        if ($.inArray(e.keyCode, [46, 8, 9, 27, 13, 110, 190]) !== -1 ||
             // Allow: Ctrl/cmd+A
            (e.keyCode == 65 && (e.ctrlKey === true || e.metaKey === true)) ||
             // Allow: Ctrl/cmd+C
            (e.keyCode == 67 && (e.ctrlKey === true || e.metaKey === true)) ||
             // Allow: Ctrl/cmd+X
            (e.keyCode == 88 && (e.ctrlKey === true || e.metaKey === true)) ||
             // Allow: home, end, left, right
            (e.keyCode >= 35 && e.keyCode <= 39)) {
                 // let it happen, don't do anything
                 return;
        }
        // Ensure that it is a number and stop the keypress
        if ((e.shiftKey || (e.keyCode < 48 || e.keyCode > 57)) && (e.keyCode < 96 || e.keyCode > 105)) {
            e.preventDefault();
        }
    });
});

使用这个DOM

<input type='text' onkeypress='validate(event)' />

和这个脚本

function validate(evt) {
  var theEvent = evt || window.event;
  var key = theEvent.keyCode || theEvent.which;
  key = String.fromCharCode( key );
  var regex = /[0-9]|\./;
  if( !regex.test(key) ) {
    theEvent.returnValue = false;
    if(theEvent.preventDefault) theEvent.preventDefault();
  }
}
