    • 是否有一种简单的方法可以用PHP从数组中删除一个元素，这样不再包含那个元素？

    我认为将它设置为null会做到这一点，但显然这是行不通的。

    删除一个数组元素有不同的方法，其中一些比其他的更有用于某些特定的任务。

    Delete one array element

    如果你只想删除一个数组元素，你可以使用 unset （）或替代 ` array_splice（ ）。

    另外，如果你有这个价值，不知道删除元素的关键，你可以使用 array_search（）来获取密钥。

    method

    注意当你使用unset（）时，数组键不会改变/ reindex。如果你想重新索引密钥，你可以使用 array_values（） unset（）之后的<> 将把所有的键转换为从0开始的数字枚举键。

    代码

    <?php

    $array = array(0 => "a", 1 => "b", 2 => "c");
    unset($array[1]);
               //a Key which you want to delete

?>

    输出

    Array (
    [0] => a
    [2] => c
)

    method

    如果使用array_splice（），键将被自动重新索引，但是关联键不会改变与 array_values（）`相反，这将把所有键转换为数字密钥。

    另外array_splice（）需要offset，而不是key !,作为第二个参数。

    代码

    <?php

    $array = array(0 => "a", 1 => "b", 2 => "c");
    array_splice($array, 1, 1);
                       //a Offset which you want to delete

?>

    输出

    Array (
    [0] => a
    [1] => c
)

    array_splice（）与unset（）一样，通过引用获取数组，这意味着你不想把这些函数的返回值赋给数组

    Delete multiple array elements

    如果您想要删除多个数组元素而不想多次调用unset（）或array_splice（），可以使用array_diff （）或array_diff_key（），具体取决于您是否知道要删除的元素的值或键。

    method

    如果您知道要删除的数组元素的值，则可以使用array_diff（）。与之前的unset（）一样，它不会改变/重新索引数组的键。

    代码

    <?php

    $array = array(0 => "a", 1 => "b", 2 => "c");
    $array = array_diff($array, ["a", "c"]);
                              //aaaaaaaaaaa Array values which you want to delete

?>

    输出

    Array (
    [1] => b
)

    method

    如果您知道要删除的元素的键，那么您要使用array_diff_key（）。在这里，你必须确保你把键作为第二个参数的键而不是值。否则，您必须使用[ array_flip（） < / A>。而且在这里键不会改变/ reindex。

    代码

    <?php

    $array = array(0 => "a", 1 => "b", 2 => "c");
    $array = array_diff_key($array, [0 => "xy", "2" => "xy"]);
                                   //a           a Array keys which you want to delete
?>

    输出

    Array (
    [1] => b
)

    另外，如果您想使用unset（）或array_splice（）删除多个具有相同值的元素，您可以使用 array_keys（）获取特定值的所有关键字，然后删除所有元素

    应该指出的是， unset（）将保持索引不变，这是当使用字符串索引（数组作为散列表）时，你会期待什么，但是在处理整数索引数组时可能会感到非常惊讶：
