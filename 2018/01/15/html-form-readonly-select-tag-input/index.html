HTML表单只读选择标记/输入

By zhizi

Published 2018-01-15

Contents

根据HTML规范，HTML中的select标记没有readonly属性，只有disabled属性。所以，如果你想让用户不用改变下拉菜单，你必须使用disabled。

唯一的问题是禁用的HTML表单输入不会包含在POST / GET数据中。

模拟select标记的readonly属性的最佳方法是什么，仍然可以获取POST数据？

您应该保持select元素禁用，但是也要添加另一个隐藏的输入，它们具有相同的名称和值。

如果您重新启用SELECT，则应将其值复制到onchange事件中的隐藏输入中，并禁用（或删除）隐藏的输入。

这是一个演示：

$('#mainform').submit(function() {
    $('#formdata_container').show();
    $('#formdata').html($(this).serialize());
    return false;
});

$('#enableselect').click(function() {
    $('#mainform input[name=animal]')
        .attr("disabled", true);

    $('#animal-select')
        .attr('disabled', false)
        .attr('name', 'animal');

    $('#enableselect').hide();
    return false;
});


#formdata_container {
    padding: 10px;
}


<script src="https://ajax.googleapis.com/ajax/libs/jquery/2.1.1/jquery.min.js"></script>
<div>
    <form id="mainform">
        <select id="animal-select" disabled="true">
            <option value="cat" selected>Cat</option>
            <option value="dog">Dog</option>
            <option value="hamster">Hamster</option>
        </select>
        <input type="hidden" name="animal" value="cat"/>
        <button id="enableselect">Enable</button>

        <select name="color">
            <option value="blue" selected>Blue</option>
            <option value="green">Green</option>
            <option value="red">Red</option>
        </select>

        <input type="submit"/>
    </form>
</div>

<div id="formdata_container" style="display:none">
    <div>Submitted data:</div>
    <div id="formdata">
    </div>
</div>

我们也可以使用这个

禁用除选定选项之外的全部功能：

<select>
<option disabled="disabled">1</option>
<option selected="selected">2</option>
<option disabled="disabled">3</option>
</select>

通过这种方式，下拉菜单仍然有效（并提交其值），但用户无法选择其他值。
