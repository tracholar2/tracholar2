如何为网站添加浏览器选项卡图标（favicon）？

By zhizi

Published 2018-01-15

Contents
  1. 1. <link rel="icon">
  2. 2. favicon.ico in the root directory
    1. 2.1. See also:

我一直在网站上工作，我想在浏览器标签上添加一个小图标。

如何在HTML中执行此操作，以及代码中需要放置的位置（例如标题）？我有一个.png徽标文件，我想将其转换为图标。

相关：浏览器选项卡上的HTML设置图片。

实际上有两种方法可以为网站添加一个图标。

<link rel="icon">

只需将以下代码添加到＆lt; head＆gt;元素：

<link rel="icon" href="http://example.com/favicon.png">

PNG图标受到大多数浏览器的支持，但IE <= 10 除外。为了向后兼容，您可以使用ICO图标。

请注意， 不必在rel属性中使用shortcut 前面的图标。从MDN链接类型：

The shortcut link type is often seen before icon, but this link type is
non-conforming, ignored and web authors must not use it anymore.

favicon.ico in the root directory

通过另一个SO回答（@麦卡托）：

All modern browsers (tested with Chrome 4, Firefox 3.5, IE8, Opera 10 and
Safari 4) will always request a favicon.ico unless you’ve specified a
shortcut icon via <link>.

所以你所要做的就是让你的网站的/favicon.ico请求返回你的图标。不幸的是，这个选项不允许你使用PNG图标。

See also:

  • favicon.png vs favicon.ico - 为什么我应该使用PNG而不是ICO？

  • 使用工具将您的PNG转换为ico文件。你可以搜索“favicon generator”，你可以找到许多在线工具。

  • 将head中的地址放在link -tag：

    ＆lt; link rel =“快捷图标”href =“http://sstatic.net/stackoverflow/img/favicon.ico”＆gt;
