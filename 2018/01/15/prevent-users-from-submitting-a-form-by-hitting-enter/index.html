通过点击Enter阻止用户提交表单

By zhizi

Published 2018-01-15

Contents
  1. 1. Disallow enter key anywhere
  2. 2. Allow enter key on textareas only
  3. 3. Allow enter key on textareas and submit buttons only

我在一个网站上进行了一项调查，似乎有一些用户打了一个问题（我不知道为什么），而不用点击提交按钮就意外地提交调查表（表单）。有没有办法来防止这一点？

我在调查中使用HTML，PHP 5.2.9和jQuery。

您可以使用诸如

的方法

$(document).ready(function() {
  $(window).keydown(function(event){
    if(event.keyCode == 13) {
      event.preventDefault();
      return false;
    }
  });
});

在阅读关于原文的评论时，为了使其更易于使用，并允许用户在完成所有字段时按Enter：

function validationFunction() {
  $('input').each(function() {
    ...

  }
  if(good) {
    return true;
  }
  return false;
}

$(document).ready(function() {
  $(window).keydown(function(event){
    if( (event.keyCode == 13) && (validationFunction() == false) ) {
      event.preventDefault();
      return false;
    }
  });
});

Disallow enter key anywhere

如果您的表单中没有＆lt; textarea＆gt;，则只需将以下内容添加到＆lt; form＆gt;：

<form ... onkeypress="return event.keyCode != 13;">

或者用jQuery：

$(document).on("keypress", "form", function(event) { 
    return event.keyCode != 13;
});

这将导致在表单中的每个按键将在keyCode上被检查。如果它不是13（Enter键），那么它将返回true，任何事情都会如预期的那样发生。如果是13（回车键），那么它将返回false，任何东西都会立即停止，所以表单不会被提交。

keypress 事件优于 keydown ，因为只有在实际插入字符时才会触发该事件。当 _任何_ 键（包括控制键）被按下时，keydown （和 keyup ）会被触发。而且， keypress 的 keyCode
表示正在插入的实际字符，而不是使用的物理键。这样你就不需要明确地检查Numpad Enter键（108）是否被按下了。 keyup
`太晚了，无法阻止表单提交。

请注意，$（window）在keydown / <代码> keypress / keyup
`，所以如果你想覆盖那些不好的用户，那么这不是一个好的选择。

Allow enter key on textareas only

如果您的表单中有＆lt; textarea＆gt;（当然， _应接受Enter键），则将按键处理程序添加到每个单独的输入元素不是＆lt; textarea＆gt;。

<input ... onkeypress="return event.keyCode != 13;">
<select ... onkeypress="return event.keyCode != 13;">
...

为了减少样板，最好用jQuery来完成：

$(document).on("keypress", ":input:not(textarea)", function(event) {
    return event.keyCode != 13;
});

如果在这些输入元素上还附加了其他事件处理函数，而您也希望在某些原因上输入密钥，那么只能防止事件的默认行为，而不是返回false，以便它可以正确传播给其他处理程序

$(document).on("keypress", ":input:not(textarea)", function(event) {
    if (event.keyCode == 13) {
        event.preventDefault();
    }
});

Allow enter key on textareas and submit buttons only

如果您希望在提交按钮＆lt; input | button type =“submit”＆gt;上允许回车键，那么您总是可以按照以下方式细化选择器。

$(document).on("keypress", ":input:not(textarea):not([type=submit])", function(event) {
    // ...
});

请注意，在某些其他答案中建议的input [type = text]并不包括那些HTML5非文本输入，所以这不是一个好的选择器。
