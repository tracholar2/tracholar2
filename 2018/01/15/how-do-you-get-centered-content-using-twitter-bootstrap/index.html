如何使用Twitter Bootstrap获得中心内容？

By zhizi

Published 2018-01-15

Contents
  1. 1. This is for Text Centering ( which is what the question was about )

我试图遵循一个非常基本的例子。使用起始页面和网格系统，我希望以下内容：

<div class="row">
  <div class="span12">
    <h1>Bootstrap starter template</h1>
    <p>Example text.</p>
  </div>
</div>

…会产生居中的文字。

不过，它仍然出现在最左边。我做错了什么？

This is for Text Centering ( which is what the question was about )

对于其他类型的内容，请参阅 Flavien的回答。

更新：Bootstrap 2.3.0+答案**

最初的答案是早期版本的bootstrap。从bootstrap 2.3.0开始， 可以简单地给div class.text-center
。

原始答案（2.3.0之前版本）

您需要使用text-align：center来定义两个类中的一个，row或span12。请参阅http://jsfiddle.net/xKSUH/ 或http://jsfiddle.net/xKSUH/1/

注意：这是在Bootstrap 3中删除。

在Bootstrap 3之前，你可以像这样使用CSS class pagination-centered：

<div class="span12 pagination-centered">
    Centered content.
</div>

Class 以分页为中心的已经在bootstrap.css（或bootstrap.min.css）中，并且只有一个规则：

.pagination-centered{text-align:center;}

Bootstrap 2.3.0。 只要使用class text-center
