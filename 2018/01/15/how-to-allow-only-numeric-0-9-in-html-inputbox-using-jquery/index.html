如何只允许使用jQuery的HTML输入框中的数字（0-9）？

By zhizi

我正在创建一个网页，其中有一个输入文本字段，我只想让数字字符（如0,1,2,3,4,5 … 9）0-9。

我怎样才能使用jQuery来做这件事？

看看这个[插件（texotela的数字jquery插件的分叉）一>。
这个（jStepper）是另外一个。

$(document).ready(function() {
    $("#txtboxToFilter").keydown(function (e) {
        // Allow: backspace, delete, tab, escape, enter and .
        if ($.inArray(e.keyCode, [46, 8, 9, 27, 13, 110, 190]) !== -1 ||
             // Allow: Ctrl+A, Command+A
            (e.keyCode === 65 && (e.ctrlKey === true || e.metaKey === true)) || 
             // Allow: home, end, left, right, down, up
            (e.keyCode >= 35 && e.keyCode <= 40)) {
                 // let it happen, don't do anything
                 return;
        }
        // Ensure that it is a number and stop the keypress
        if ((e.shiftKey || (e.keyCode < 48 || e.keyCode > 57)) && (e.keyCode < 96 || e.keyCode > 105)) {
            e.preventDefault();
        }
    });
});

注意：如果您的网页使用HTML5，您可以使用内置的＆lt; input type =“number”＆gt;，并使用min和max属性来控制最小值和最大值。

以下是我使用的功能：

// Numeric only control handler
jQuery.fn.ForceNumericOnly =
function()
{
    return this.each(function()
    {
        $(this).keydown(function(e)
        {
            var key = e.charCode || e.keyCode || 0;
            // allow backspace, tab, delete, enter, arrows, numbers and keypad numbers ONLY
            // home, end, period, and numpad decimal
            return (
                key == 8 || 
                key == 9 ||
                key == 13 ||
                key == 46 ||
                key == 110 ||
                key == 190 ||
                (key >= 35 && key <= 40) ||
                (key >= 48 && key <= 57) ||
                (key >= 96 && key <= 105));
        });
    });
};

然后，您可以通过以下操作将其附加到您的控件中：

$("#yourTextBoxName").ForceNumericOnly();
