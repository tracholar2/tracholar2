CSS中心文本（水平和垂直）在一个div块内

我将div设置为display：block。 （90px height和width），并且里面有一些文字。

我需要将文字垂直和水平对齐。

我已经尝试了text-align：center，但是它没有执行水平部分，所以我尝试了vertical-align：middle，但是没有成功。

有什么想法？

如果是一行文字和/或图像，那么很容易做到。只需使用

text-align: center;
vertical-align: middle;
line-height: 90px;       /* the same as your div height */

就是这样。如果它可以是多行，那么它更复杂一些。但在 http://pmob.co.uk/上有解决方案。寻找“垂直对齐”。

因为他们往往是黑客或添加复杂的div …我通常使用一个单一的单元格表来做到这一点…尽可能简单。

Update for 2016 / 2017:

使用transform可以更经常地完成，甚至在IE10和IE11等旧版浏览器中也能正常工作。它可以支持多行文本：

position: relative;
top: 50%;
transform: translateY(-50%);

例如： https://jsfiddle.net/wb8u02kL/1/

To shrink wrap the width:

上面的解决方案使用固定宽度的内容区域。要使用收缩包装宽度，请使用

position: relative;
float: left;
top: 50%;
left: 50%;
transform: translate(-50%, -50%);

例如： https://jsfiddle.net/wb8u02kL/2/

我尝试了flexbox，但并没有得到广泛的支持，因为它会在IE10的一些老版本IE中崩溃。

Common techniques as of 2014:

方法1 - transform translateX / translateY：

这里的示例 / 全屏示例

在支持的浏览器（大部分）中，您可以使用top：50％ / left：50％与translateX（-50％）translateY（-50％）结合动态垂直/水平居中元素。

      .container {
    位置：绝对;
    顶部：50％;
    剩下：50％;
    转换：translateX（-50％）translateY（-50％）;
}

  • 方法2 - Flexbox方法：

此处示例 / 全屏示例

在支持的浏览器中，将目标元素的display设置为flex并使用align-items：center进行垂直居中，justify-content：center进行水平居中。请不要忘记添加供应商前缀以获取其他浏览器支持（查看示例）。

      html，body，.container {
    身高：100％
}
。容器 {
    显示：flex;
    align-items：center;
    justify-content：center;
}

方法3 - table-cell / vertical-align：middle：

此处示例 / 全屏示例

在某些情况下，您需要确保html / body元素的高度设置为100％。

对于垂直对齐，将父元素的width / height设置为100％并添加display：table 。然后对于子元素，将display更改为table-cell并添加vertical-align：middle。

对于水平居中，您可以添加text-align：center以将文本和任何其他inline子元素居中。或者，您可以使用margin：0 auto假设元素是block级别。

      html，body {
    身高：100％
}
.parent {
    宽度：100％;
    身高：100％
    显示：表格;
    text-align：center;
}
.parent＆gt; .child {
    display：table-cell;
    vertical-align：middle;
}

方法4 - 绝对定位50％从顶部排列：

这里的示例 / 全屏示例

这种方法假设文本具有已知的高度，在这种情况下是18px。只需将元素50％从顶部相对于父元素绝对定位即可。在这种情况下，请使用元素已知高度的一半的负值 margin-top 值 -9px `。

      html，body，.container {
    身高：100％
}
。容器 {
    位置：相对;
    text-align：center;
}
.container＆gt; p {
    位置：绝对;
    顶部：50％;
    左：0;
    正确：0;
    margin-top：-9px;
}

方法5 - line-height方法（最不灵活 - 不建议）：

此处示例

在某些情况下，父元素将有一个固定的高度。对于垂直居中，您所要做的就是在子元素上设置一个line-height值，该值等于父元素的固定高度。

虽然这个解决方案在某些情况下可以工作，但是值得注意的是，当有多行文本时，它不会工作 -
像这样。

      .parent {
    height：200px;
    宽度：400px;
    text-align：center;
}
.parent＆gt; .child {
    line-height：200px;
}

方法4和5不是最可靠的。去前3个之一。
