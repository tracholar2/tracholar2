我怎样才能把我的div放在容器的底部？

By zhizi

Published 2018-01-15

Contents

给出以下HTML：

<div id="container">
  <!-- Other elements here -->
  <div id="copyright">
    Copyright Foo web designs
  </div>
</div>

我希望#copyright粘贴到#container的底部。

我可以做到这一点，而不使用绝对定位？如果浮动属性支持“底部”的值，似乎会做的伎俩，但不幸的是，它不会。

可能不是。

将 position：relative 分配给 #container ，然后 position：absolute;底部：0; 到#copyright `。

*


#container {

}
#copyright {
position: absolute;
bottom: 0;
}



  <div id="copyright">
    Copyright Foo web designs
  </div>
</div>

实际上，只有当窗口很高，内容很短时，@
User才能接受的答案。但是，如果内容较高且窗口较短，则会在页面内容上放置版权信息，然后向下滚动以查看内容将为您留下浮动版权声明。这使得这个解决方案对于大多数页面（如这个页面，实际上）都是没有用的。

这样做的最常见的方式是“CSS粘脚”的方法，或略微更小的变化。这种方法很好 - 如果你有一个固定的高度页脚。

如果你需要一个可变高度的页脚，如果内容太短，会出现在窗口的底部，如果窗口太短，在内容的底部，你会怎么做？

吞下你的骄傲，并使用一张桌子。

例如：

* {
    padding: 0;
    margin: 0;
}

html, body {
    height: 100%;
}

#container {
    height: 100%;
    border-collapse: collapse;
}


<!DOCTYPE html>
<html>
<body>
    <table id="container">
        <tr>
            <td valign="top">
                <div id="main">Lorem ipsum, etc.</div>
            </td>
        </tr>
        <tr>
            <td valign="bottom">
                <div id="footer">Copyright some evil company...</div>
            </td>
        </tr>
    </table>
</body>
</html>

试试吧。这将适用于任何窗口大小，任何数量的内容，任何大小的页脚，每个浏览器…甚至IE6。

如果你想要使用表格进行布局，请花点时间问自己为什么。 CSS本该让我们的生活变得更加简单 - 总体来说 -
但事实是，即使在这么多年之后，它仍然是一个破坏的，反直觉的混乱。它不能解决所有问题。这是不完整的。

表格并不酷，但至少现在，它们有时是解决设计问题的最好方法。
