如何设置输入type =“file”按钮的样式。

造型文件输入是非常困难的，因为大多数浏览器不会改变外观从CSS或JavaScript。

即使输入的大小也不会回应如下的内容：

<input type="file" style="width:200px">

相反，您将需要使用size属性：

<input type="file" size="60" />

对于任何比这更复杂的样式（例如，改变浏览按钮的外观），您将需要考虑在本机文件输入之上重叠样式化按钮和输入框的技巧方法。 rm在www.quirksmode.org/dom/inputfile.html中已经提到的文章是我见过的最好的一个。

You don’t need JavaScript for this! Here is a cross-browser solution:

查看这个例子！ - 它在Chrome / FF / IE中运行 -
（IE10 / 9/8/7）

最好的方法是将一个自定义标签元素与一个属性连接到一个 隐藏 文件输入元素。 （标签的for属性必须与文件元素的id匹配才能工作）。

<label for="file-upload" class="custom-file-upload">
    Custom Upload
</label>
<input id="file-upload" type="file"/>

另外，也可以直接用文本标签来包装文件输入元素： **（example） < /强> 

<label class="custom-file-upload">
    <input type="file"/>
    Custom Upload
</label>

在样式方面，只需使用 1 输入元素， rel =“noreferrer”>属性选择器。

input[type="file"] {
    display: none;
}

然后，你需要做的是定制标签元素。 （example） 。

.custom-file-upload {
    border: 1px solid #ccc;
    display: inline-block;
    padding: 6px 12px;
    cursor: pointer;
}

- 值得注意的是，如果你使用display：none来隐藏该元素，那么在IE8及其以下版本中将不起作用。也要注意的事实，即jQuery验证不验证隐藏字段默认。如果这些事情中的任何一个都是你的问题，这里有两种不同的方法来隐藏输入（ 1， 2）在这些情况下工作。
