使用CSS内容添加HTML实体

By zhizi

Published 2018-01-15

Contents

你如何使用CSS content属性添加html实体？

我有许多链接，通过在每个链接之前添加一个箭头，将其制作成面包屑样式列表。

<div class="breadcrumbs">
    <a>One</a>
    <a>Two</a>
    <a>Three</a>
</div>

具有以下风格：

.breadcrumbs a:before {
    content: '> ';
}

问题是当面包屑列表变长并换行到下一行时，箭头留在前面的行。显而易见的解决方案是将空间更改为不间断的空间，但这不起作用：

.breadcrumbs a:before {
    content: '>&nbsp;';
}

它实际上输出＆amp; nbsp;到屏幕上。我通过直接粘贴字符来解决其他奇怪的字符（如»），但是如何粘贴＆amp; nbsp;？

你必须使用转义的unicode：

像

.breadcrumbs a:before {
    content: '>\0000a0';
}

更多信息： http：/ /www.evotech.net/blog/2007/04/named-html-entities-in-numeric-
order/

CSS不是HTML。 ＆amp; nbsp;是HTML中的字符实体引用;字符引用＆amp;＃160;的替代品。 160是 Unicode 中的NO-BREAK SPACE字符的十进制 代码点 ，UCS-2 ;请参阅HTML
4.01规范）。该代码点的十六进制表示是U + 00A0（160 = 10×16 + 0×16 0 ）。你会发现，在Unicode
代码图表和字符数据库。

在CSS中，您需要使用Unicode转义序列来表示这些字符，这是基于字符代码点的十六进制值。所以你需要写

.breadcrumbs a:before {
  content: '>\a0';
}

只要转义序列的最后一个字符串的值是这样的。如果有字符出现，有两种方法可以避免误解：

a）（被他人提及）使用正好六个十六进制数字作为转义序列：

.breadcrumbs a:before {
  content: '>\0000a0foo';
}

b）在转义序列之后添加一个空格（例如空格）：

.breadcrumbs a:before {
  content: '>\a0 foo';
}

（因为f是一个十六进制数字，所以\ a0f在这里是指GURMUKHI LETTER EE，或者如果你有合适的字体。）

分隔的空格将被忽略，并将显示＆gt; foo，其中显示的空格将是一个NO-BREAK SPACE字符。

空格方法（'\ a0 foo'）比六位方法（'\ 0000a0foo'）具有以下优点：

  • 键入**比较容易，因为前面的零不是必须的，数字也不需要计算;
  • 阅读起来比较容易 ，因为转义序列和后面的文本之间有空白，数字不需要计算;
  • 占用更少的空间 ，因为前导零不是必需的;
  • 向上兼容 ，因为将来Unicode支持超出U + 10FFFF的代码点将需要修改CSS规范。

因此，要在转义字符后面显示一个空格，请在样式表中使用 两个 空格 -

.breadcrumbs a:before {
  content: '>\a0  foo';
}

- 或者明确指出：

.breadcrumbs a:before {
  content: '>\a0\20 foo';
}

有关详细信息，请参见[ CSS 2.1，“4.1.3个性和案例”部分](http://www.w3.org/TR/CSS2/syndata.html

#escaped-characters)
