将div垂直居中在另一个div内

By zhizi

发表于 2018-01-15

文章目录

我想将一个div添加到另一个div中。

<div id="outerDiv">
    <div id="innerDiv">
    </div>
</div>

这是我目前使用的CSS。

#outerDiv{
    width: 500px;
    height: 500px;
    position:relative;
}

#innerDiv{
    width: 284px;
    height: 290px;
    position:absolute;
    top: 50%;
    left:50%;
    margin-top: -147px;
    margin-left: -144px;
}

正如你所看到的，我现在使用的方法取决于innerDiv的宽度和高度的值。如果宽度/高度改变，我将不得不修改margin-top < / code>和 margin-left values.Is有任何通用的解决方案，我可以用来居中 innerDiv `总是不管它的大小吗？

我发现使用margin：auto可以将内部的div水平地对齐到中间。但是纵向排列中间呢？

tl; dr

垂直对齐中间的作品，但是你必须在子元素上使用table-cell和inline-block。

这个解决方案在IE6＆amp; 7.你的是更安全的方式。
但是因为你用CSS3和HTML5标记了你的问题，我以为你不介意使用现代的解决方案。

经典的解决方案（表格布局）**

这是我原来的答案。它仍然正常工作，是最广泛支持的解决方案。表格式将影响你的渲染性能，所以我建议你使用更现代的解决方案之一。

下面是一个例子

测试：**

  • FF3.5 +
  • FF4 +
  • Safari 5 +
  • Chrome 11 +
  • IE9 +

<强> HTML

<div class="cn"><div class="inner">your content</div></div>

<强> CSS

.cn {
  display: table-cell;
  width: 500px;
  height: 500px;
  vertical-align: middle;
  text-align: center;
}

.inner {
  display: inline-block;
  width: 200px; height: 200px;
}

现代解决方案（转换）**

由于现在得到了很好的支持，所以有一种更简单的方法。 / p>

<强> CSS

.cn {
  position: relative;
  width: 500px;
  height: 500px;
}

.inner {
  position: absolute;
  top: 50%; left: 50%;
  transform: translate(-50%,-50%);
  width: 200px;
  height: 200px;
}

演示

我最喜欢的现代解决方案（flexbox）**

我开始越来越多地使用flexbox
现在也得到了很好的支持它是迄今为止最简单的方法

<强> CSS

.cn {
  display: flex;
  justify-content: center;
  align-items: center; 
}

演示

更多例子＆amp;可能性： 比较一个网页上的所有方法

实现这种水平和垂直居中的另一种方法是：

.Absolute-Center {
  margin: auto;
  position: absolute;
  top: 0; left: 0; bottom: 0; right: 0;
}
