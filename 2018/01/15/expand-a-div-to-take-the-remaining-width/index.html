展开一个div来取其余的宽度

By zhizi

Published 2018-01-15

Contents

我想要一个双列的div布局，其中每个布局可以有不同的宽度，例如

div {
    float: left;
}
.second {
    background: #ccc;
}


<div>Tree</div>
<div class="second">View</div>

我希望’view’div扩展到’tree’div填充需要的空间之后可用的整个宽度。 目前我的’视图’div被调整到它包含的内容
如果两个div都占据整个高度，那也是好的

不得重复声明：

浮动时将div展开为最大宽度：设置为left 因为那里有一个固定的宽度。

[帮助div - 使div适合其余宽度< / A> 因为我需要两列都对齐到左边

对此的解决方案其实很简单，但并不是所有的 __
显而易见的。你必须触发一个叫做“块格式化上下文”（BFC）的东西，它以一种特殊的方式与浮点数交互。

取第二个div，删除浮点数，然后给它overflow：hidden来代替。除可见性之外的任何溢出值都会使其设置的块成为BFC。
BFC不允许后裔浮游物逃离它们，也不允许兄弟/祖先浮游物闯入它们。这里的净效果是浮动的div会做的事情，然后第二个div将是一个普通的块，占用所有可用的宽度
除了浮动占用 。

这应该适用于所有当前的浏览器，尽管您可能必须在IE6和7中触发hasLayout。我不记得。
