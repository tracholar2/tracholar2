如何使用PHP获取客户端IP地址？

我想保留通过他/她的IP地址登录到我的网站的用户的记录。

不管你做什么，都要确保不要信任从客户端发送的数据。 $ _ SERVER ['REMOTE_ADDR']包含连接方的真实IP地址。这是你可以找到的最可靠的价值。

有几个HTTP头像X-Forwarded-For，这些头可以或不可以由各种代理设置。问题是这些只是HTTP头可以由任何人设置。他们的内容不能保证。
$ _ SERVER ['REMOTE_ADDR']是Web服务器从中接收连接的实际物理IP地址，并且该响应将被发送到。其他任何事情都只是武断的和自愿的信息。只有一种情况下，您可以信任这些信息：您正在控制设置此标头的代理。意思是只有当你知道100％在哪里以及如何设置标题时，你才会注意到它的重要性。

话虽如此，下面是一些示例代码：

if (!empty($_SERVER['HTTP_CLIENT_IP'])) {
    $ip = $_SERVER['HTTP_CLIENT_IP'];
} elseif (!empty($_SERVER['HTTP_X_FORWARDED_FOR'])) {
    $ip = $_SERVER['HTTP_X_FORWARDED_FOR'];
} else {
    $ip = $_SERVER['REMOTE_ADDR'];
}

编者注： 使用上面的代码有 安全隐患 。客户端可以将所有的HTTP头信息（即。$ _ SERVER ['HTTP _...））设置为任意值。因此，使用$ _ SERVER ['REMOTE_ADDR']更加可靠，因为这不能由用户设置。
