把一个div变成一个链接

By zhizi

Published 2018-01-15

Contents

我有一个＆lt; div＆gt;块，其中包含一些我不想改变的花哨的视觉内容。我想将其设为可点击的链接。

我正在寻找类似于＆lt; a href =“...＆gt;＆lt; div＆gt; ＆lt; / div＆gt;＆lt; / a＆gt;，但这是有效的XHTML 1.1。

来到这里希望找到一个更好的解决方案，但我不喜欢这里提供的任何一个。我想你们有些人误解了这个问题。
OP想让一个充满内容的行为像一个链接。其中一个例子就是Facebook广告 - 如果你看，他们实际上是正确的标记。

对于我来说，这个no-nos是：javascript（不应该只是一个链接，非常糟糕的搜索引擎优化/可访问性）;无效的HTML。

本质上是这样的：

  • 使用普通的CSS技术和有效的HTML构建面板。
  • 在那里放置一个链接，如果用户点击面板，您希望成为默认链接（也可以有其他链接）。
  • 在该链接中，放置一个空的span标记（＆lt; span＆gt;＆lt; / span＆gt;，而不是＆lt; span /＆gt; - 谢谢@Campey） / LI>
  • 给出面板位置：相对
  • 将下面的CSS应用于空白区域：

<预> <代码> { 位置：绝对的; 宽度：100％; 高度：100％; 顶部：0; 左：0; z-index：1; /
修复IE7 / 8中的重叠错误， 确保你有一个空的gif / background-image：url（’empty.gif’）; }

现在它将覆盖面板，并且它位于＆lt; A＆gt;标记内，是一个可点击的链接

  • 给出面板位置内的任何其他链接：相对和一个合适的z-索引（＆gt; 1），使它们位于默认跨度链接之前

您不能将div本身作为链接，但可以使＆lt; a＆gt;标记充当block ＆lt; div＆gt;具有相同的行为。

a {
    display: block;
}

然后你可以设置它的宽度和高度。
