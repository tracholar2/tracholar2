如何去除内联块元素之间的空间？

By zhizi

发表于 2018-01-15

文章目录

鉴于这个HTML和CSS：

span {
    display:inline-block;
    width:100px;
    background-color:palevioletred;
}


<p>
    <span> Foo </span>
    <span> Bar </span>
</p>

因此，SPAN元素之间会有一个4像素宽的空间。

演示： http://jsfiddle.net/dGHFV/ < / p>
我明白为什么会发生这种情况，而且我也知道可以通过消除HTML源代码中的SPAN元素之间的空白来消除该空间，如下所示：

<p>
    <span> Foo </span><span> Bar </span>
</p>

然而，我希望有一个不需要HTML源代码被篡改的CSS解决方案。

我知道如何通过JavaScript解决这个问题 - 通过从容器元素（段落）中删除文本节点，如下所示：

// jQuery
$('p').contents().filter(function() { return this.nodeType === 3; }).remove();

演示 ：[ http://jsfiddle.net/dGHFV/1/ < / A>

但是这个问题可以用CSS来解决吗？

由于这个答案已经变得相当流行，我正在重写它。

我们不要忘记被问到的实际问题：

How to remove the space between inline-block elements? I was hoping for
a CSS solution that doesn’t require the HTML source code to be tampered with.
Can this issue be solved with CSS alone?

可以用CSS来解决这个问题，但是没有完全可靠的CSS修正。

我最初的答案是在父元素中添加font-size：0，然后在children上声明一个合理的font-size。

http://jsfiddle.net/thirtydot/dGHFV/1361/

这在所有现代浏览器的最新版本中都适用。它在IE8中工作。它在Safari 5中不起作用，但是它在Safari 6中起作用。Safari
5几乎是一个死的浏览器（ 0.33％，2015年8月）。

相对字体大小的大部分可能的问题并不复杂，以解决。

然而，如果你只是特别需要一个CSS修正，这是一个合理的解决方案，但如果你可以自由地改变你的HTML（我们大部分人都是这样），这不是我所推荐的。

/ p>

这就是我作为一名经验丰富的网页开发人员，实际上可以解决这个问题：

<p>
    <span>Foo</span><span>Bar</span>
</p>

是的，没错。我在内嵌块元素之间的HTML中删除空格。
