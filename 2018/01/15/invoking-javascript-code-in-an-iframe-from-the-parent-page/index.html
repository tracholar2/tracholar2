在父页面的iframe中调用JavaScript代码

By zhizi

Published 2018-01-15

Contents

基本上，我有一个嵌入在页面中的iframe，iframe有一些 JavaScript例程需要从父页面调用。

现在，相反很简单，因为你只需要调用parent.functionName（），但不幸的是我需要完全相反的。

请注意，我的问题并未改变iframe的来源网址，而是调用函数在iframe中定义。

假设你的iFrame的id是“targetFrame”，你想调用的函数是targetFunction（）：

document.getElementById('targetFrame').contentWindow.targetFunction();

您也可以使用window.frames而不是document.getElementById来访问框架。

// this option does not work in most of latest versions of chrome and Firefox
window.frames[0].frameElement.contentWindow.targetFunction();

在这里有一些怪癖。

  • 完全有可能是小孩的iframe还没有加载，或者是别的什么地方出错让它无法访问。您可能会发现更容易反转通信流程：也就是说，当子结束加载并准备好回调时，让子iframe通知其window.parent脚本。通过将自己的一个对象（例如一个回调函数）传递给父脚本，父母就可以直接与iframe中的脚本进行通信，而不必担心它与之相关的HTMLIFrameElement。
