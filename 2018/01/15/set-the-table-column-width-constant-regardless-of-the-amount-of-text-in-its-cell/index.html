无论单元格中文本的数量如何，表格列的宽度是不变的？

By zhizi

Published 2018-01-15

Contents

在我的表中，我将列中第一个单元格的宽度设置为100px。
但是，如果此列中某个单元格中的文本太长，则列的宽度将超过100px。我怎么能禁用这个扩展？

我玩了一下，因为我很难搞清楚。

您需要设置单元格宽度（th或td工作，我同时设置）并将table-layout设置为fixed。出于某种原因，单元格宽度似乎只能保持固定，如果表宽度也设置（我认为这是愚蠢的，但whatev）。将溢出属性设置为隐藏也很有用。

你应该确保离开CSS的所有边界和大小 好的，这就是我所拥有的。

HTML：

<table>
   <tr>
      <th>header 1</th>
      <th>header 234567895678657</th>
   </tr>
   <tr>
      <td>data asdfasdfasdfasdfasdf</td>
      <td>data 2</td>
   </tr>
</table>

CSS：

table{
    border: 1px solid black;
    table-layout: fixed;
    width: 200px;
}

th, td {
    border: 1px solid black;
    width: 100px;
}
