改变hr元素的颜色

By zhizi

发表于 2018-01-15

文章目录

我想用CSS改变我的hr标签的颜色。我在下面尝试的代码似乎不工作：

hr {
  color: #123455;
}

如果你的目的是改变＆lt; hr＆gt;所产生的行的颜色，我想你应该使用 border-color 来代替 color ` /
code>标记。

虽然已经指出，如果你改变了行的大小，边框的宽度仍然和你在样式中所指定的一样宽，并且行会被默认的颜色填充（这不是一个理想的效果大多数时候）。所以在这种情况下，你也需要指定background-color（@Ibu在他的回答中提出）。

HTML 5 Boilerplate项目在其默认样式表中**指定以下规则：

hr {
    display: block;
    height: 1px;
    border: 0;
    border-top: 1px solid #ccc;
    margin: 1em 0;
    padding: 0; 
}
