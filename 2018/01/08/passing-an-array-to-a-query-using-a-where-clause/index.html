使用WHERE子句将数组传递给查询

By zhizi

发表于 2018-01-08

文章目录

给定一个ID数组$galleries = array(1,2,5)我想有一个SQL查询，在其WHERE子句中使用数组的值，如：

SELECT *
FROM galleries
WHERE id = /* values of array $galleries... eg. (1 || 2 || 5) */

我怎样才能生成这个查询字符串用于MySQL？

BEWARE! This answer contains a severe SQL
injection vulnerability. Do NOT use the code samples as presented
here, without making sure that any external input is sanitized.

$ids = join("','",$galleries);   
$sql = "SELECT * FROM galleries WHERE id IN ('$ids')";

使用PDO：[1]

$in = join(',', array_fill(0, count($ids), '?'));
$select = <<<SQL
    SELECT *
    FROM galleries
    WHERE id IN ($in);
SQL;
$statement = $pdo->prepare($select);
$statement->execute($ids);

使用MySQLi [2]

$in = join(',', array_fill(0, count($ids), '?'));
$select = <<<SQL
    SELECT *
    FROM galleries
    WHERE id IN ($in);
SQL;
$statement = $mysqli->prepare($select);
$statement->bind_param(str_repeat('i', count($ids)), ...$ids);
$statement->execute();
$result = $statement->get_result();

说明：

Use the SQL IN() operator to check if a value exists in a given list.

一般来说，它看起来像这样：

expr IN (value,...)

我们可以建立一个表达式放置在我们数组中的()中。请注意，括号内必须至少有一个值，否则MySQL将返回一个错误;这相当于确保我们的输入数组至少有一个值。为了防止SQL注入攻击，首先为每个输入项目生成?以创建参数化查询。这里我假设包含你的id的数组叫做$ids：

$in = join(',', array_fill(0, count($ids), '?'));

$select = <<<SQL
    SELECT *
    FROM galleries
    WHERE id IN ($in);
SQL;

给定三个项目的输入数组$select将如下所示：

SELECT *
FROM galleries
WHERE id IN (?, ?, ?)

再次注意，输入数组中的每个项目都有一个?。然后，我们将使用PDO或MySQLi来准备和执行上面提到的查询。

Using the IN() operator with strings

由于绑定的参数，在字符串和整数之间很容易改变。对于PDO，不需要改变;对于MySQLi，如果需要检查字符串，请将str_repeat('i',更改为str_repeat('s',。

[1]：为了简洁，我省略了一些错误检查。您需要检查每个数据库方法的常见错误（或者将您的数据库驱动程序设置为抛出异常）。

[2]：需要PHP 5.6或更高版本。为了简洁，我再次省略了一些错误检查。
