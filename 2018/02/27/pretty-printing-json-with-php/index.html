漂亮的打印与PHP的JSON

By zhizi

发表于 2018-02-27

文章目录

我正在构建一个将JSON数据提供给另一个脚本的PHP脚本。我的脚本将数据构建到一个大的关联数组中，然后使用json_encode输出数据。这是一个脚本示例：

$data = array('a' => 'apple', 'b' => 'banana', 'c' => 'catnip');
header('Content-type: text/javascript');
echo json_encode($data);

上面的代码产生以下输出：

{"a":"apple","b":"banana","c":"catnip"}

如果你有少量的数据，这是非常好的，但我更喜欢这些方面的内容：

{
    "a": "apple",
    "b": "banana",
    "c": "catnip"
}

有没有办法在PHP中做到这一点，而不是丑陋的黑客？看起来好像 Facebook上的人发现了这一点。

PHP 5.4提供了用于json_encode（）调用的JSON_PRETTY_PRINT选项。

http://php.net/manual/en/function.json- encode.php

<?php
...
$json_string = json_encode($data, JSON_PRETTY_PRINT);

这个函数将采用JSON字符串并缩进它非常可读。它也应该收敛，

prettyPrint( $json ) === prettyPrint( prettyPrint( $json ) )

输入

{"key1":[1,2,3],"key2":"value"}

输出

{
    "key1": [            1,
        2,
        3
    ],
    "key2": "value"
}

代码

function prettyPrint( $json )
{
    $result = '';
    $level = 0;
    $in_quotes = false;
    $in_escape = false;
    $ends_line_level = NULL;
    $json_length = strlen( $json );

    for( $i = 0; $i < $json_length; $i++ ) {
        $char = $json[$i];
        $new_line_level = NULL;
        $post = "";
        if( $ends_line_level !== NULL ) {
            $new_line_level = $ends_line_level;
            $ends_line_level = NULL;
        }
        if ( $in_escape ) {
            $in_escape = false;
        } else if( $char === '"' ) {
            $in_quotes = !$in_quotes;
        } else if( ! $in_quotes ) {
            switch( $char ) {
                case '}': case ']':
                    $level--;
                    $ends_line_level = NULL;
                    $new_line_level = $level;
                    break;

                case '{': case '[':
                    $level++;
                case ',':
                    $ends_line_level = $level;
                    break;

                case ':':
                    $post = " ";
                    break;

                case " ": case "\t": case "\n": case "\r":
                    $char = "";
                    $ends_line_level = $new_line_level;
                    $new_line_level = NULL;
                    break;
            }
        } else if ( $char === '\\' ) {
            $in_escape = true;
        }
        if( $new_line_level !== NULL ) {
            $result .= "\n".str_repeat( "\t", $new_line_level );
        }
        $result .= $char.$post;
    }

    return $result;
}
