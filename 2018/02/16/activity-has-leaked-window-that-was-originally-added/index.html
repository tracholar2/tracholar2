这个问题是谷歌Android开发人员最热门的搜索之一， 因此，从评论中加入少数重要的观点，这可能对未来的调查人员更有帮助，而不会深入地进行评论交流。

答案1 ：

You’re trying to show a Dialog after you’ve exited an Activity.

答案2

This error can be a little misleading in some circumstances (although the
answer is still completely accurate) - i.e. in my case an unhandled Exception
was thrown in an AsyncTask, which caused the Activity to shutdown, then an
open progressdialog caused this Exception.. so the ‘real’ exception was a
little earlier in the log

答案3

Call dismiss() on the Dialog instance you created before exiting your
Activity, e.g. in onPause() or onDestroy()

解决方法是在退出Activity之前，在 viewP.java:183 中创建的 Dialog 上调用 dismiss（），例如在onPause（）中。所有Window s＆amp; Dialog应在离开Activity之前关闭。
