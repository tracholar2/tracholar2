我在哪里可以放置在Android Studio中的“资产”文件夹？

By zhizi

Published 2018-02-01

Contents

我对assets文件夹感到困惑。它不会在Android Studio中自动创建，几乎所有讨论此问题的论坛都会讨论Eclipse。

如何在Android Studio中配置资产目录

由于Android Studio使用新的基于Gradle的构建系统，因此您应该是将assets /放入源代码集（例如src / main / assets /）。

在典型的Android Studio项目中，您将拥有一个带有main /源码集（app / src / main /）的app />关闭项目根目录），所以你的主要资产将进入app / src / main / assets /。但是：

如果您需要特定于构建的资产（如debug vs release），则可以为这些角色创建源代码集（例如，` app / SRC /释放/资产/
）

  • 您的产品口味也可以包含资源的源代码集（例如app / src / googleplay / assets /）

您的工具测试可以有一个带有自定义资源的androidTest源码集（例如app / src / androidTest / assets /），但是一定要问getContext（），而不是getTargetContext（）的InstrumentationRegistry来访问这些资源

另外，还要提醒一下：资产在运行时是只读的。使用内部存储，外部存储，或者存储访问框架读取/写入内容。

让Android Studio为您完成。_ **

  • 在Android Studio（ 1.0＆amp; above ）中，右键点击文件夹并导航到 Assets Folder 。

  1. 在下一个屏幕上，点击 完成 。

瞧！它将在 main 目标源集中创建 assets 文件夹。
