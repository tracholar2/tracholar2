Facebook API - 如何通过Facebook API获取Facebook用户的个人资料图片（无需用户“允许”该应用程序）

By zhizi

Published 2018-05-15

Contents
  1. 1. 50x50 pixels
  2. 2. 200 pixels width
  3. 3. To save (using PHP)

我正在制作一个内容管理系统，用于从他们的Facebook
URL （也就是http://facebook.com/users_unique_url）。我怎样才能做到这一点？是否有Faceboook
API调用来获取用户的个人资料图片URL，而用户不需要 允许 应用程序？

只需通过网址获取数据：

http://graph.facebook.com/userid_here/picture

将userid_here替换为要获取照片的用户的ID。您也可以使用HTTPS。

您可以使用PHP file_get_contents函数读取该URL并处理检索的数据。

<强>资源：

http://developers.facebook.com/docs/api

注意：**在php.ini中，您需要确保 OpenSSL扩展可以使用PHP的file_get_contents功能读取该URL。

显示：

50x50 pixels

<img src="//graph.facebook.com/{{fid}}/picture">

200 pixels width

<img src="//graph.facebook.com/{{fid}}/picture?type=large">

To save (using PHP)

注意： 不要使用此 。请参阅下面的@ Foreever的评论。

$img = file_get_contents('https://graph.facebook.com/'.$fid.'/picture?type=large');
$file = dirname(__file__).'/avatar/'.$fid.'.jpg';
file_put_contents($file, $img);

其中$ fid是Facebook上的用户标识（或昵称）。

注意：如果图像标记为“18+”，则需要来自18位以上用户的有效access_token：

<img src="//graph.facebook.com/{{fid}}/picture?access_token={{access_token}}">
