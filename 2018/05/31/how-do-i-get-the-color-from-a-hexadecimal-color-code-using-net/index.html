如何从十六进制颜色代码（例如＃FFDFD991）获取颜色？

我正在阅读一个文件，并得到一个十六进制颜色代码。我需要为十六进制颜色代码创建相应的System.Windows.Media.Color实例。在框架中是否有内置的方法？

我假设这是一个ARGB代码…你指的是System.Drawing.Color或System.Windows.Media.Color？后者用于WPF中。我还没有看到任何人提到它，以防万一你正在寻找它：

using System.Windows.Media;

Color color = (Color)ColorConverter.ConvertFromString("#FFDFD991");

假设您是指HTML类型的RGB代码（称为十六进制代码，例如＃FFCC66），请使用 ColorTranslator class：

System.Drawing.Color col = System.Drawing.ColorTranslator.FromHtml("#FFCC66");

如果您使用ARGB十六进制代码，则可以使用 ColorConverter 来自System.Windows.Media命名空间的类：

Color col = ColorConverter.ConvertFromString("#FFDFD991") as Color;
//or      = (Color) ColorConverter.ConvertFromString("#FFCC66") ;
