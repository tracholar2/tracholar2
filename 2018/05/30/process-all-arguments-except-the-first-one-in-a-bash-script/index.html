处理除第一个之外的所有参数（在bash脚本中）

By zhizi

Published 2018-05-30

Contents

我有一个简单的脚本，其中第一个参数是为文件名保留的，所有其他可选参数都应该传递给脚本的其他部分。

使用Google，我找到了这个wiki ，但是它提供了一个字面的例子：

echo "${@: -1}"

我无法得到任何其他的工作，如：

echo "${@:2}"

或

echo "${@:2,1}"

我从终端上得到“坏取代”。

问题是什么，如何处理除了传递给bash脚本的第一个参数？

使用这个：

echo "${@:2}"

以下语法：

echo "${*:2}"

也可以，但不推荐，因为[ @ Gordon已经解释过，使用*，它将所有参数作为一个单独的参数与空格一起运行，而@保留它们之间的中断（即使一些参数本身包含空格）。它与echo没有什么不同，但是对于其他许多命令来说，它是很重要的。

如果您想要一个解决方案，也可以在/ bin / sh中尝试

first_arg="$1"
shift
echo First argument: "$first_arg"
echo Remaining arguments: "$@"

shift [n] 移动位置参数 _n_ 次。 shift 将 $ 1 的值设置为 $ 2 的值， $ 2 的值设置为<代码> $ 3等，将$＃的值减1。
