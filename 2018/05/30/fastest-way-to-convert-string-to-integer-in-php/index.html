使用PHP，将字符串转换为整数的最快方法是：“123”？

为什么特定的方法最快？如果得到意外的输入，如“hello”或一个数组

，会发生什么？

我刚刚建立了一个快速的基准测试：

附录 ：我刚刚遇到一个稍微出乎意料的行为，您应该在选择其中一种方法时注意到这一点：

$x = "11";
(int) $x;      // int(11)
intval($x);    // int(11)
$x + 0;        // int(11)

$x = "0x11";
(int) $x;      // int(0)
intval($x);    // int(0)
$x + 0;        // int(17) !

$x = "011";
(int) $x;      // int(11)
intval($x);    // int(11)
$x + 0;        // int(11) (not 9)

使用PHP 5.3.1测试

我个人觉得演员是最漂亮的。

$iSomeVar = (int) $sSomeOtherVar;

如果发送像“Hello”这样的字符串，它将被转换为整数0.对于像“22岁”这样的字符串，它将被转换为整数22.任何它不能解析的数字变成0

如果你真的需要这个速度的话，我猜这里的其他建议是正确的，假设强制是最快的。
