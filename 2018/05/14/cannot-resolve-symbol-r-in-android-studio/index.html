Android Studio中的“无法解析符号R”

By zhizi

Published 2018-05-14

Contents

在所有引用R.id.something的类中的每一个实例中，R都是红色的，它说“无法解析符号R”。也每次有R.layout.something红色下划线，并说“无法解决方法setContentView（？）”。该项目总是很好。看到这一直是令人讨厌的。我已经在这里阅读了许多其他类似的问题，但大部分都涉及从eclipse导入项目。我正在使用我认为是Android工作室的最新版本，该项目是用android工作室创建的，没有任何解决R问题的工作。如果有人知道，我想知道是什么原因造成的。

<强>更新： 解决方案：当时android
studio是全新的，我也是一个全新的开发者。我应该从来没有使用过Android工作室，我从来没有意识到如何未完成。感谢大家的支持，因为这已经发布。具有讽刺意味的是，这个问题的实际回答已被删除，不能被取消删除。我觉得我原来的答案是解决这个问题的方法。

我原来的回答被删除了：

I think I have figured out the issue. The issue is with Android Studio,
every time I added the admob library all these errors started. All my other
projects without ads don’t have similar issues. I’m now using Intelij and
staying away from Android Studio

我也有这个问题。一个简单的“gra clean清理”和“gra build建立”就完成了这个任务。

点击 Build-＆gt;清理项目 ，即可执行gradle clean

在最新版本的Android Studio中，至少对我而言，以下几个作品：

“工具” - ＆gt; “Android” - ＆gt; “与Gradle文件同步项目”
