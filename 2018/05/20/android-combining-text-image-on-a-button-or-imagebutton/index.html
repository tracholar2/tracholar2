Android：在Button或ImageButton上结合文本和图像

By zhizi

发表于 2018-05-20

文章目录

我试图在一个按钮上添加一个图像（作为背景），并根据运行时发生的情况动态添加一些图片上方/上方的文字。

如果我使用ImageButton，我甚至没有可能添加文本。 如果我使用Button，我可以添加文本，但只能用android：drawableBottom和定义的类似XML属性定义一个图像这里。

但是，这些属性只能结合文本和文本。图像在x和y维度，这意味着我可以在我的文字周围画一个图像，但不能在我的文本下面/下面（z轴被定义为从显示屏出来）。

任何建议如何做到这一点？一个想法是扩展Button或者ImageButton，并覆盖draw（） -
方法。但是用我目前的知识水平，我并不知道如何去做（2D渲染）。也许有更多经验的人知道一个解决方案，或者至少有一些指针开始？

您可以在Button上调用setBackgroundDrawable（）来设置按钮的背景。

任何文字都会显示在背景上方。

如果你正在寻找类似的XML有： android：background属性的工作方式相同。

对于只想将背景，图标图像和文本放在一个** Button中的用户，可以使用不同的文件：设置在Button背景上，drawableTop /
Bottom / Rigth / Left和 padding 属性。

<Button
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:background="@drawable/home_btn_test"
        android:drawableTop="@drawable/home_icon_test"
        android:textColor="#FFFFFF"
        android:id="@+id/ButtonTest"
        android:paddingTop="32sp"
        android:drawablePadding="-15sp"
        android:text="this is text"></Button>

对于更复杂的安排，您也可以使用RelativeLayout并使其可点击。
