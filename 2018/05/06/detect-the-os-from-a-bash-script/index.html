从Bash脚本中检测操作系统

By zhizi

发表于 2018-05-06

文章目录

我想在版本控制中保留我的.bashrc和.bash_login文件，以便我可以在所有使用的计算机之间使用它们。问题是我有一些操作系统特定的别名，所以我正在寻找一种方法来确定脚本是否运行在Mac OS
X，Linux或 Cygwin的。

在 Bash脚本中检测操作系统的正确方法是什么？

对于我的.bashrc，我使用下面的代码：

platform='unknown'
unamestr=`uname`    if [[ "$unamestr" == 'Linux' ]]; then
   platform='linux'
elif [[ "$unamestr" == 'FreeBSD' ]]; then
   platform='freebsd'
fi

然后我做一些事情：

if [[ $platform == 'linux' ]]; then
   alias ls='ls --color=auto'
elif [[ $platform == 'freebsd' ]]; then
   alias ls='ls -G'
fi

这是丑陋的，但它的作品。如果您喜欢，可以使用case来代替if。

**bash manpage说变量OSTYPE存储操作系统的名字：

OSTYPE Automatically set to a string that describes the operating system
on which bash is executing. The default is system- dependent.

这里设置为linux-gnu。
