如何查看Android Studio中的方法信息？

By zhizi

发表于 2018-05-29

文章目录

在Eclipse中，当你将鼠标悬停在某个方法上时，将会出现一个窗口，其中包含了该方法的功能描述，参数的含义以及返回的内容。有没有办法让Android
Studio做同样的事情？

最简单最直接的方式：

您可以通过以下方式激活它： 文件＆gt;设置＆gt;编辑器＆gt;常规 ，然后选中 在移动鼠标时显示快速文档 ：

[![Settings dialog with checked

option](https://i.stack.imgur.com/boacD.png)](https://i.stack.imgur.com/boacD.png)

其他方式：

  • 您可以进入IntelliJ的bin文件夹并搜索 idea.properties 。将这一行添加到文档中：

<代码> auto.show.quick.doc =真

现在，您将拥有与Eclipse相同的浮动文档窗口。

  • 您必须按CTRL + Q查看Javadoc。

您可以固定窗口，并且每次使用鼠标选择方法时都会显示文档。

Android Studio 1.0： 如果您想要获取文档窗口，请持有CTRL。否则当您将鼠标从方法文档窗口移开时，滚动文档将消失。
