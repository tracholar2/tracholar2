在PHPMyAdmin中导入文件大小限制

By zhizi

发表于 2018-05-28

文章目录

我已经改变了我所知道的所有php.ini参数： upload_max_filesize，post_max_size。

为什么我仍然看到2MB？

使用Zend Server CE，在Windows 7主机上的Ubuntu VirtualBox上。

您可能没有重新启动您的服务器;）

或者你修改了错误的php.ini。

或者你真的设法做两个^^

在服务器上找到名为php.ini的文件，然后按照以下步骤操作

安装apache2和php5你需要在php.ini文件中进行三处更改。首先打开文件进行编辑，例如：

sudo gedit /etc/php5/apache2/php.ini
OR

sudo gedit /etc/php/7.0/apache2/php.ini
接下来，搜索post_max_size条目，然后输入比数据库大小（本例中为15M）更大的数字，例如：

post_max_size = 25M

接下来编辑memory_limit的条目，并给它一个比给 max_size `更大的值。

然后确保upload_max_filesize的值小于post_max_size。

从最大到最小的顺序应该是：

memory_limit
post_max_size
upload_max_filesize

保存文件后，重新启动apache（例如sudo /etc/init.d/apache2 restart）并设置好。

不要忘记重新启动Apache服务以应用更改。
