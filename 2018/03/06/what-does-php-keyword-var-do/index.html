这是为了在PHP4中声明类成员变量，不再需要。它将在PHP5中工作，但是会在PHP中从5.0.0版本到5.1.2版本引发E_STRICT警告，因为它不被使用。自PHP 5.3以来，var一直没有被弃用，并且是“public”的同义词。

示例用法：

class foo {
    var $x = 'y'; // or you can use public like...
    public $x = 'y'; //this is also a class member variables.
    function bar() {
    }
}

var关键字用于声明变量PHP 4中的一个类：

class Foo {
    var $bar;
}

使用PHP 5
属性和方法可见性（，protected和private），因此不推荐使用var。
