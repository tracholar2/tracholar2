我试图将日期从yyyy-mm-dd转换为dd-mm-yyyy（但不是在SQL中）;但是我不知道日期函数如何需要时间戳，我不能从这个字符串获得时间戳。

这怎么可能？

使用strtotime（）和date（）：

$originalDate = "2010-03-21";
$newDate = date("d-m-Y", strtotime($originalDate));

（请参阅 strtotime和日期文档在PHP网站上）。

编辑**：请注意，这是对原始问题的快速解决方案。对于更广泛的转换，您应该使用 DateTime类来解析＆amp;格式： - ）

如果你想避免strtotime转换（例如strtotime不能解析你的输入），你可以使用，

$myDateTime = DateTime::createFromFormat('Y-m-d', $dateString);
$newDateString = $myDateTime->format('d-m-Y');

或等同于：

$newDateString = date_format(date_create_from_format('Y-m-d', $dateString), 'd-m-Y');

你首先给它的格式是$ date String。然后你告诉它你希望$ new Date String的格式。

如果源格式总是“Ymd”（yyyy-mm-
dd），那么只需使用日期时间：

<?php
$source = '2012-07-31';
$date = new DateTime($source);
echo $date->format('d.m.Y'); // 31.07.2012
echo $date->format('d-m-Y'); // 31-07-2012
?>

?>
