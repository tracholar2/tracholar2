逐行读取文件，将该值分配给变量

By zhizi

Published 2018-03-28

Contents

我有以下的.txt文件：

Marco
Paolo
Antonio

我想逐行读取它，并且为每一行我想分配一个.txt行值到一个变量。假设我的变量是$ name，流程是：

  • 阅读文件的第一行
  • 分配$ name =“Marco”

  • 使用$ name完成一些任务

  • 阅读文件第二行

  • 分配$ name =“Paolo”

以下内容（保存为rr.sh）读取作为参数逐行传递的文件：

#!/bin/bash
while IFS='' read -r line || [[ -n "$line" ]]; do
    echo "Text read from file: $line"
done < "$1"

说明：

  • IFS =“（或IFS =）可以防止前/后的空格被剪裁。
  • -r可防止解释反斜杠转义。
  • <代码> ||如果它不以\ n结尾，那么[[-n $ line]] 可以防止最后一行被忽略（因为 read `遇到EOF时为零退出代码）。

运行脚本如下：

chmod +x rr.sh
./rr.sh filename.txt

…

我鼓励你为read使用-r标志，`代表：

-r  Do not treat a backslash character in any special way. Consider each
    backslash to be part of the input line.

我引用man 1 read。

另一件事是把文件名作为参数。

这是更新后的代码：

#!/usr/bin/bash
filename="$1"
while read -r line
do
    name="$line"
    echo "Name read from file - $name"
done < "$filename"
