您可能需要将时区放在php.ini文件的配置行中。你应该在你的php.ini文件中有这样的块：

[Date]
; Defines the default timezone used by the date functions
; http://php.net/date.timezone
date.timezone = America/New_York

如果不是，请添加它（以您的时区替换时区）。配置完成后，请确保重新启动httpd（service httpd restart）。

以下是支持的时区列表。

如果你不能修改你的php.ini配置，你可以在你的代码开始处使用下面的代码片段：

date_default_timezone_set('Africa/Lagos');//or change to whatever timezone you want

时区列表可以在 http://www.php.net/manual/en/timezones .PHP。
