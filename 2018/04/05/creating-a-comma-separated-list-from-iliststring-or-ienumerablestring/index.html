从IList string或IEnumerable string创建逗号分隔的列表

By zhizi

发表于 2018-04-05

文章目录

从` IList＆lt; string＆gt;＆lt; / code＆gt;或＆lt; code＆gt; IEnumerable＆lt;
string＆gt;＆lt; / code＆gt;创建逗号分隔的字符串值列表的最简单方法是什么？

String.Join（...）在string []上运行，因此当IList＆lt; string＆gt; / code>或IEnumerable＆lt; string＆gt; `无法轻易转换为字符串数组。

.NET 4 + **

IList<string> strings = new List<string>{"1","2","testing"};
string joined = string.Join(",", strings);

细节＆amp; Pre .Net 4.0解决方案

使用LINQ（.NET 3.5）很容易将IEnumerable＆lt; string＆gt; `转换为字符串数组 _：

IEnumerable<string> strings = ...;
string[] array = strings.ToArray();

如果您需要：编写等效的帮助程序方法很容易：

public static T[] ToArray(IEnumerable<T> source)
{
    return new List<T>(source).ToArray();
}

然后像这样调用它：

IEnumerable<string> strings = ...;
string[] array = Helpers.ToArray(strings);

然后你可以调用string.Join。当然，你不用_来使用辅助方法：

// C# 3 and .NET 3.5 way:
string joined = string.Join(",", strings.ToArray());
// C# 2 and .NET 2.0 way:
string joined = string.Join(",", new List<string>(strings).ToArray());

后者虽然有点含糊不清：）

这很可能是最简单的方法，而且性能也是非常好的 - 还有其他的问题，比如（但不限于） .com / questions / 219519 / whats-
the-c-method-syntax-for-converting-an-array-to-a-simple-string“>这个。

从.NET 4.0开始，在 中有更多的重载> string.Join ，所以你可以只写：

string joined = string.Join(",", strings);

更简单：）

仅供参考，.NET 4.0版本的string.Join（）有一些额外的重载，它与IEnumerable一起使用，而不仅仅是数组，包括可以处理任何类型的T：

public static string Join(string separator, IEnumerable<string> values)
public static string Join<T>(string separator, IEnumerable<T> values)
