C＃循环 - 休息与继续

By zhizi

Published 2018-04-04

Contents

在C＃中（随意回答其他语言）循环，break和continue之间的区别是什么，作为离开循环结构的一种手段，并且继续下一次迭代？

示例：

foreach (DataRow row in myTable.Rows)
{
    if (someConditionEvalsToTrue)
    {
        break; //what's the difference between this and continue ?
        //continue;
    }
}

break将完​​全退出循环，continue将会 跳过 当前迭代。

例如：

for (int i = 0; i < 10; i++) {
    if (i == 0) {
        break;
    }

    DoSomeThingWith(i);
}

break会导致循环在第一次迭代时退出 - DoSomeThingWith永远不会被执行。这里：

for (int i = 0; i < 10; i++) {
    if(i == 0) {
        continue;
    }

    DoSomeThingWith(i);
}

不会为i = 0执行DoSomeThingWith，但循环会 继续 和DoSomeThingWith对于i = 1to i = 9。

理解这个的一个非常简单的方法是在每个关键字后面放置“loop”这个词。现在的条款是有道理的，如果他们只是像日常短语一样阅读。

` 循环被打破并停止。

继续` **循环继续执行下一次迭代。
