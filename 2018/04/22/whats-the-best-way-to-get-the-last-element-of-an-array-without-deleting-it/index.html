获取数组的最后一个元素而不删除它的最好方法是什么？

By zhizi

Published 2018-04-22

Contents

确定，

我完全了解 array_pop（），但是删除了最后一个元件。获取数组的最后一个元素而不删除它的最好方法是什么？

编辑：这是一个奖金：

$array = array('a' => 'a', 'b' => 'b', 'c' => 'c');

甚至

$array = array('a', 'b', 'c', 'd');
unset($array[2]);
echo $array[sizeof($array) - 1]; // Output: PHP Notice:  Undefined offset:  2 in - on line 4

短而甜。

我想出了解决方案来删除错误信息并保留单行表单和高效的性能：

$lastEl = array_values(array_slice($array, -1))[0];

- 以前的解决方案

$lastEl = array_pop((array_slice($array, -1)));

注意：需要额外的括号以避免出现PHP严格标准：只有变量应该通过引用传递。

试试

$myLastElement = end($yourArray);

重置它（谢谢@hopeseekr）：

reset($yourArray);

链接到手册

@David Murdoch补充道： $ myLastElement = end（array_values（$ yourArray））; //现在你不需要调用reset（）。 在E_STRICT上会产生警告

Strict Standards: Only variables should be passed by reference
