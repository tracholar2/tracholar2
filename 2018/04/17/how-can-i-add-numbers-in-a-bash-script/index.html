如何在bash脚本中添加数字

By zhizi

发表于 2018-04-17

文章目录

我有这个bash脚本，并且在第16行有一个问题。 我怎样才能把第15行的以前的结果，并添加 它到第16行的变量？

#!/bin/bash

num=0
metab=0

for ((i=1; i<=2; i++)); do      
    for j in `ls output-$i-*`; do
        echo "$j"

        metab=$(cat $j|grep EndBuffer|awk '{sum+=$2} END { print sum/120}') (line15)
        num= $num + $metab   (line16)
    done
    echo "$num"
 done

_ ：

  • 使用算术扩展：$（（EXPR）） 
    num = $（（num1 + num2））
    num = $（（$ num1 + $ num2））＃也可以
    num = $（（num1 + 2 + 3））＃…
    num = $ [num1 + num2]＃旧的，不推荐的算术表达式语法

  • 使用外部的expr工具。请注意，这只适用于真正旧的系统。

    num =`expr $ num1 + $ num2`＃expr的空格是很重要的

浮点 ：

Bash不直接支持这个，但是有几个外部工具可以使用：

num=$(awk "BEGIN {print $num1+$num2; exit}")
num=$(python -c "print $num1+$num2")
num=$(perl -e "print $num1+$num2")
num=$(echo $num1 + $num2 | bc)   # whitespace for echo is important

您也可以使用科学记数法（例如：2.5e + 2）

常见的陷阱_：

  • 设置变量时，不能在=的任一侧留下空格，否则会迫使shell将第一个单词解释为要运行的应用程序的名称（例如： num =或num）

~~num = 1 ~~ ~~num = 2 ~~

bc 和 expr 期望每个数字和运算符都是一个单独的参数，所以空白是很重要的。他们无法处理像 3 + +4 `这样的参数。

~~num =expr $ num1 + $ num2 ~~

使用$（（））算术扩展。

num=$(( $num + $metab ))

请参阅 http://tldp.org/LDP/abs/html/arithexp.html以获取更多信息。
