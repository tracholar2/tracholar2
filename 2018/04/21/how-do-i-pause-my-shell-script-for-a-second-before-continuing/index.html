在继续之前，如何暂停我的shell脚本？

By zhizi

Published 2018-04-21

Contents

我只找到了如何等待用户输入。不过，我只想暂停，这样我的while true不会使我的计算机崩溃。

我尝试了pause（1），但是在意外的标记’1’附近写了-bash：syntax error。怎么做呢？

使用sleep命令。

示例：

sleep .5 # Waits 0.5 second.
sleep 5  # Waits 5 seconds.
sleep 5s # Waits 5 seconds.
sleep 5m # Waits 5 minutes.
sleep 5h # Waits 5 hours.
sleep 5d # Waits 5 days.

在Python中（问题最初是标记为Python），您需要导入时间模块

import time
time.sleep(1)

或

from time import sleep
sleep(1)

对于shell脚本来说就是

sleep 1

执行sleep命令。例如。 / bin / sleep
