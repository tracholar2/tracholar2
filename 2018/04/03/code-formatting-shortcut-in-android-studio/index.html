Android studio中的代码格式化快捷方式

By zhizi

Published 2018-04-03

Contents
  1. 1. EDIT

我已经开始使用 Android Studio进行开发。在eclipse中我使用的是Ctrl + Shift + F，但是在Android studio中它不起作用，它会有所不同。

如何跳转到.java中的任何方法。我在eclipse中使用Ctrl + O。

可用吗？任何人都可以告诉我这个代码格式的快捷方式吗？

运

Ctrl + Alt + L
Linux的：

Ctrl + Shift + Alt + L
的Mac：

Option + Command + L
参考：关键命令，这里是完整的命令 Windows /
Linux用户和Mac用户

更新为Ubuntu 由于Rohit在Ubuntu中面临的问题是格式化代码快捷方式，因为在Ubuntu中使用Ctrl + Alt + L 键来锁定屏幕。

我发现Ubuntu首先处理重要事件。所以试着改变Ubuntu中的Ctrl + Alt + L动作。

步骤是：_ **

转到系统工具 - ＆gt;系统设置 - ＆gt;键盘 - ＆gt;快捷键标签 - ＆gt;系统 - ＆gt;锁定屏幕
选择行New Accelerator ...将会显示，然后按任意带有Alpha键的特殊键（如Shift + L）。您更改了快捷键。

现在检查Studio中的格式键。

或者

Ctrl + Alt + Shift + L (for a dialog)

更新来自亨利庄，也在用户2340612给出的另一个答案。

**EM>

您也可以使用Eclipse快捷方式。 Windows和Linux，请转到File＆gt;设置＆gt; Keymap，Mac OS X，转到＆gt;
首选项＆gt; Keymap，然后从下拉菜单中选择Eclipse。

您也可以使用Eclipse的键盘快捷键：只需进入 preferences-＆gt; keymap ，然后从下拉菜单中选择 eclipse 。

EDIT

实际路径是：File-＆gt; Settings-＆gt;（在IDE设置下） - ＆gt; Keymap（感谢@Dhinakaran
