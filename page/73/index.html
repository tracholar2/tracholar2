Xmas95 = new Date("20 09 2006,12:52:39")

在调用new Date（）函数时，我无法在任何地方找到显示所有有效字符串格式的文档。

这是为了将字符串转换为日期。如果我们看另一面，即将日期对象转换为字符串，直到现在我还是觉得JavaScript没有内置API来将日期对象格式化为字符串。

Editor’s note: The following approach is the asker’s attempt that worked
on a particular browser but does not work in general; see the answers on
this page to see some actual solutions.

今天，我在date对象上使用了toString（）方法，并且令人惊讶地发挥了将日期格式化为字符串的作用。

var d1 = new Date();
d1.toString('yyyy-MM-dd');       //Returns "2009-06-29" in Internet Explorer, but not Firefox or Chrome
d1.toString('dddd, MMMM ,yyyy')  //Returns "Monday, June 29,2009" in Internet Explorer, but not Firefox or Chrome

另外在这里，我找不到任何有关我们可以将日期对象格式化为字符串的方法的文档。
