这个例子来自文档：

class C(object):
    def __init__(self):
        self._x = None

    def getx(self):
        return self._x
    def setx(self, value):
        self._x = value
    def delx(self):
        del self._x
    x = property(getx, setx, delx, "I'm the 'x' property.")

property的参数是getx，setx，delx和一个doc字符串。

在下面的代码中，property用作装饰器。它的对象是x函数，但是在上面的代码中，参数中没有对象函数的地方。

class C(object):

    def x(self):

    @x.setter
    def x(self, value):
        self._x = value

    @x.deleter
