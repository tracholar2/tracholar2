  HashMap<String, String> theHash = (HashMap<String, String>)session.getAttribute("attributeKey");
  return theHash;
}

如果可能的话，我想避免Eclipse警告，因为理论上它们至少表示一个潜在的代码问题。尽管如此，我还没有找到消除这个问题的好方法。我可以将涉及到的单行提取到一个方法本身，并向该方法添加@SuppressWarnings（“unchecked”），从而限制了代码块在忽略警告的情况下的影响。有更好的选择吗？我不想在Eclipse中关闭这些警告。
