让div填充剩余的屏幕空间的高度

By zhizi

Published 2018-01-15

我目前正在开发一个Web应用程序，我希望内容填满整个屏幕的高度。

该页面有一个标题，其中包含一个标志和帐户信息。这可能是一个任意的高度。我想要内容div来填充页面的其余部分。

我有一个标题div和一个内容div。目前，我正在使用如下表格的布局：

CSS和HTML

#page {
    height: 100%; width: 100%
}

#tdcontent {
    height: 100%;
}

#content {
    overflow: auto; /* or overflow: hidden; */
}


<table id="page">
    <tr>
        <td id="tdheader">
            <div id="header">...</div>
        </td>
    </tr>
    <tr>
        <td id="tdcontent">
            <div id="content">...</div>
        </td>
    </tr>
</table>

页面的整个高度被填满，不需要滚动。

对于内容div中的任何内容，设置top：0;会将它放在标题的正下方。有时候，内容将是一个真正的表格，其高度设置为100％。将header放在content里面不会允许这个工作。

有没有一种方法可以在不使用table？的情况下达到相同的效果？

<强>更新

内容div中的元素的高度也会设置为百分比。因此，在div内部有100％的内容会填充到底部。至于两个元素将达到50％。

更新2：**

例如，如果标题占用屏幕高度的20％，那么在#content内指定为50％的表将占用屏幕空间的40％。到目前为止，把整个东西包装在一个表中是唯一有效的方法。
