property装饰器是如何工作的？

By zhizi

Published 2018-01-14

我想了解内置函数property是如何工作的。我感到困惑的是，property也可以作为装饰器使用，但是只有作为内置函数使用时才需要参数，而不能用作装饰器。

这个例子来自文档：

class C(object):
    def __init__(self):
        self._x = None

    def getx(self):
        return self._x
    def setx(self, value):
        self._x = value
    def delx(self):
        del self._x
    x = property(getx, setx, delx, "I'm the 'x' property.")

property的参数是getx，setx，delx和一个doc字符串。

在下面的代码中，property用作装饰器。它的对象是x函数，但是在上面的代码中，参数中没有对象函数的地方。

class C(object):
    def __init__(self):
        self._x = None

    @property
    def x(self):
        """I'm the 'x' property."""
        return self._x

    @x.setter
    def x(self, value):
        self._x = value

    @x.deleter
    def x(self):
        del self._x

而且，如何创建x.setter和x.deleter装饰器？ 我很困惑。
