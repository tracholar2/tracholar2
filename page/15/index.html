如何更新package.json中的每个依赖项到最新版本？

By zhizi

Published 2018-01-14

最简单的方法是什么？

我现在知道的最好的方法是运行npm info express version，然后手动更新每个package.json。必须有更好的方法。

{
  "name": "myproject",
  "description": "my node project",
  "version": "1.0.0",
  "engines": {
    "node": "0.8.4",
    "npm": "1.1.65"
  },
  "private": true,
  "dependencies": {
    "express": "~3.0.3", // how do I get these bumped to latest?
    "mongodb": "~1.2.5",
    "underscore": "~1.4.2",
    "rjs": "~2.9.0",
    "jade": "~0.27.2",
    "async": "~0.1.22"
  }
}

我现在是 npm-check-updates 的合作者，这是一个很棒的解决这个问题。
