检查一个Bash数组是否包含一个值

By zhizi

Published 2018-06-14

在Bash中，测试数组是否包含特定值的最简单方法是什么？

编辑 ：在回答和评论的帮助下，经过一些测试，我想出了这个：

function contains() {
    local n=$#
    local value=${!n}
    for ((i=1;i < $#;i++)) {
        if [ "${!i}" == "${value}" ]; then
            echo "y"
            return 0
        fi
    }
    echo "n"
    return 1
}

A=("one" "two" "three four")
if [ $(contains "${A[@]}" "one") == "y" ]; then
    echo "contains one"
fi
if [ $(contains "${A[@]}" "three") == "y" ]; then
    echo "contains three"
fi

我不确定这是否是最好的解决方案，但似乎有效。
